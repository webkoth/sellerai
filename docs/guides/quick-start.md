# 🚀 Быстрый старт

Добро пожаловать в Marketplace Workspace! Это руководство поможет начать работу за 5 минут.

## Шаг 1: Настройка API ключей

1. Получите токены в личных кабинетах:
   - **Wildberries**: [seller.wildberries.ru](https://seller.wildberries.ru/) → Настройки → Доступ к API
   
2. Скопируйте шаблон и заполните:
   ```bash
   cp config/credentials.template.json config/credentials.json
   # Отредактируйте config/credentials.json
   ```

3. Настройте MCP-серверы:
   ```bash
   cp .mcp.json.template .mcp.json
   # Замените ${WB_API_TOKEN} на реальный токен
   ```

## Шаг 2: Первые команды

Откройте Claude Code в этой директории и попробуйте:

```
/status
```
→ Покажет текущее состояние магазина

```
/digest
```
→ Утренний обзор: заказы, проблемы, рекомендации

```
/help
```
→ Список всех доступных команд

## Шаг 3: Основные сценарии

### 📊 Посмотреть продажи

## Полезные советы

### 💡 Как задавать вопросы

Claude понимает естественный язык. Примеры:

- "Какие товары заканчиваются на складе?"
- "Есть ли негативные отзывы без ответа?"
- "Покажи самые прибыльные товары за месяц"
- "Сравни мои цены с конкурентами по кроссовкам"

### ⚠️ Критичные операции

Изменение цен, остатков или рекламных бюджетов требует подтверждения. Claude всегда спросит "ДА" перед выполнением.

### 📁 Где хранятся данные

```
data/
├── cache/      ← кэш API (автоматически)
├── exports/    ← ваши выгрузки и отчёты
└── snapshots/  ← история для трендов
```
