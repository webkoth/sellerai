# 📚 Справочник команд

Все доступные slash-команды для работы с маркетплейсами.

---

## 🏠 Общие

| Команда | Описание | Пример |
|---------|----------|--------|
| `/status` | Быстрый статус магазина | `/status` |
| `/digest` | Утренний дайджест | `/digest` |
| `/help` | Список команд | `/help` |

---

## 📊 Аналитика

| Команда | Описание | Пример |
|---------|----------|--------|
| `/sales [период]` | Анализ продаж | `/sales неделя` |
| `/abc` | ABC-анализ ассортимента | `/abc` |
| `/trends` | Динамика метрик | `/trends` |
| `/competitors [артикул]` | Анализ конкурентов | `/competitors 12345` |

**Форматы периода:**
- `сегодня`, `вчера`
- `неделя`, `7д`
- `месяц`, `30д`
- `01.12-15.12` — диапазон дат

## 🎯 Подсказки

1. **Комбинируйте команды:**
   ```
   /stocks
   А теперь /forecast 30
   ```

2. **Уточняйте детали:**
   ```
   /sales месяц
   Покажи только категорию "Обувь"
   ```

3. **Просите объяснений:**
   ```
   /unit-economics 12345
   Объясни, почему такая низкая маржа
   ```
