## Как начать

1. **Выбрать тариф** — Starter для старта, Growth для роста, Scale для масштаба
2. **Предоставить доступ** — API-токены маркетплейсов (инструкция при подключении)
3. **Онбординг** — настройка за 2-3 дня, первый аудит и план действий
4. **Работа** — регулярные отчёты, оптимизация, поддержка
