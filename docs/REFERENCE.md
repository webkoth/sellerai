Skills — это модули доменных знаний, которые активируются автоматически по контексту запроса. Каждый skill содержит экспертные знания, извлечённые из курсов по маркетплейсам.

#### wb-inventory

**Путь:** `.claude/skills/wb-inventory/SKILL.md`

**Описание:** Получение остатков товаров с маркетплейса Wildberries через API.

**Триггеры:** остатки, склады, наличие, сколько товара, stock

**Возможности:**
- Получение остатков на складах FBO
- Получение цен и скидок
- Фильтрация по минимальному количеству
- Вывод в Markdown или JSON

**Скрипт:** `scripts/get_stock.py`

```bash
# Базовый запуск
python3 scripts/get_stock.py

# С параметрами
python3 scripts/get_stock.py --min-quantity 5 --warehouses
python3 scripts/get_stock.py --format json --output stock.json
```

**Требует:** `WB_API_TOKEN` с правами "Статистика" и "Цены и скидки"

#### wb-unit-economics

**Путь:** `.claude/skills/wb-unit-economics/SKILL.md`

**Описание:** Расчёт юнит-экономики товара на Wildberries.

**Триггеры:** маржа, себестоимость, прибыль, юнит-экономика, рентабельность, точка безубыточности

**Содержит:**
- Формулы расчёта маржи с учётом всех комиссий WB
- Таблица комиссий по категориям (2024-2025)
- Расчёт точки безубыточности
- ROI рекламных кампаний
- Калькулятор план/факт

**Ключевые формулы:**

```
Маржа = Цена продажи - Себестоимость - Комиссия WB - Логистика - Хранение - Эквайринг - Налоги

Маржинальность (%) = Маржа / Цена продажи × 100%

Точка безубыточности = Постоянные расходы / Маржа с единицы
```

**Комиссии WB (структура):**
| Статья | Диапазон |
|--------|----------|
| Комиссия категории | 5-15% |
| Логистика | 50-200₽ |
| Хранение | 0.5-5₽/день |
| Эквайринг | 2% |

#### wb-sales-funnel

**Путь:** `.claude/skills/wb-sales-funnel/SKILL.md`

**Описание:** Анализ воронки продаж на Wildberries.

**Триггеры:** воронка, конверсия, CTR, клики, просмотры, корзина, заказ, выкуп

**Этапы воронки:**
```
Показы → Клики → Карточка → Корзина → Заказ → Выкуп
```

**Бенчмарки конверсий:**
| Этап | Норма | Хорошо | Отлично |
|------|-------|--------|---------|
| CTR (показы→клики) | 2-3% | 4-5% | >6% |
| Карточка→корзина | 5-10% | 10-15% | >15% |
| Корзина→заказ | 20-30% | 30-40% | >40% |
| Заказ→выкуп | 50-70% | 70-85% | >85% |

**Диагностика проблем:**
| Проблема | Причина | Решение |
|----------|---------|---------|
| Низкий CTR | Плохое главное фото/цена | Улучшить фото, пересмотреть цену |
| Низкая конверсия в корзину | Слабый контент карточки | SEO, фото, описание |
| Низкий выкуп | Несоответствие ожиданиям | Точнее описание, качество |

**Типы рекламы WB:**
| Тип | Описание | Когда использовать |
|-----|----------|-------------------|
| Автореклама | Автоматический показ | Быстрый старт |
| Поиск | В результатах поиска | Целевой трафик |

| Метрика | Формула | Норма |

| CPO | Расход / Заказы | < Маржи |
| ROI | (Доход - Расход) / Расход × 100% | >100% |
| ДРР | Расход / Выручка × 100% | <10-15% |

**Стратегии ставок:**
| Цель | Стратегия | Ставка |
|------|-----------|--------|
| Запуск | Агрессивная | +20-30% от рынка |
| Удержание | Умеренная | На уровне рынка |
| Прибыль | Консервативная | Только прибыльные ключи |

#### wb-supply-chain

**Путь:** `.claude/skills/wb-supply-chain/SKILL.md`

**Описание:** Логистика и поставки на Wildberries (FBO/FBS).

**Триггеры:** поставки, склады, FBO, FBS, логистика, отгрузка, остатки на складе

**Сравнение моделей:**
| Критерий | FBO | FBS |
|----------|-----|-----|
| Хранение | На складе WB | У продавца |
| Доставка | 1-3 дня | 3-7 дней |
| Позиции в выдаче | Выше | Ниже |
| Затраты на хранение | Есть | Нет |

**Основные склады WB:**
| Склад | Регион | Покрытие |
|-------|--------|----------|
| Коледино | Московская обл. | Центр, Северо-Запад |
| Подольск | Московская обл. | Центр |
| Казань | Татарстан | Поволжье |
| Екатеринбург | Свердловская обл. | Урал |
| Новосибирск | Новосибирская обл. | Сибирь |
| Краснодар | Краснодарский край | Юг |

**Формула запаса:**
```
Запас = Средние продажи/день × Дней до поставки × Коэфф. безопасности

Коэффициенты:
- Стабильный спрос: 1.2
- Умеренные колебания: 1.5
- Высокая волатильность: 2.0
```

**Формула точки заказа:**
```
Точка заказа = Продажи/день × Срок поставки + Страховой запас
```

#### wb-pricing-strategy

**Путь:** `.claude/skills/wb-pricing-strategy/SKILL.md`

**Описание:** Ценообразование и акции на Wildberries.

**Триггеры:** цена, СПП, скидка, акция, ценообразование, распродажа

**СПП (Скидка Постоянного Покупателя) 2024:**
| Цена товара | СПП |
|-------------|-----|
| до 500₽ | 0-17% |
| 500-1000₽ | 17-25% |
| 1000-2000₽ | 25-30% |
| 2000₽+ | до 35% |

**Оптимизация СПП:**
```
Пример:
- Цена 290₽ → СПП 17% → Покупатель видит 240₽
- Цена 320₽ → СПП 25% → Покупатель видит 240₽

При одинаковой цене для покупателя выгоднее ставить 320₽!
```

**Когда участвовать в акциях:**
| Ситуация | Участвовать? |
|----------|--------------|
| Запуск товара | ✅ Да |
| Много остатков | ✅ Да |
| Падение позиций | ✅ Да |
| Пик сезона | ❌ Нет |
| Мало остатков | ❌ Нет |

**Стратегии по этапу:**
| Этап | Стратегия | Цена |
|------|-----------|------|
| Запуск | Низкая | -10-20% от рынка |
| Рост | Повышение | К рыночной |
| Зрелость | Максимум | Рыночная+ |
| Спад | Распродажа | Снижение |

## Scripts

### get_stock.py

**Путь:** `scripts/get_stock.py`

**Описание:** Получение остатков товаров через WB API

**Требует:** `WB_API_TOKEN`

**Параметры:**
| Флаг | Описание | По умолчанию |
|------|----------|--------------|
| `--format`, `-f` | Формат: markdown/json | markdown |
| `--warehouses`, `-w` | Показать склады | false |
| `--min-quantity`, `-q` | Мин. остаток | 1 |
| `--output`, `-o` | Файл для сохранения | stdout |

**Примеры:**
```bash
python3 scripts/get_stock.py
python3 scripts/get_stock.py -q 5 -w
python3 scripts/get_stock.py -f json -o stock.json
```

## Формулы и расчёты

### Юнит-экономика

```
Маржа = Цена - Себестоимость - Комиссия - Логистика - Хранение - Эквайринг - Налоги

Маржинальность = Маржа / Цена × 100%

Рентабельность = Маржа / Себестоимость × 100%

Точка безубыточности (шт) = Постоянные расходы / Маржа с единицы
```

### Воронка продаж

```
CTR = Клики / Показы × 100%

CR (конверсия) = Заказы / Клики × 100%

Выкупаемость = Выкупы / Заказы × 100%
```

### Реклама

```
CPO = Расход / Заказы

ROI = (Доход - Расход) / Расход × 100%

ROAS = Доход / Расход

ДРР = Расход / Выручка × 100%
```

### Запасы

```
Оптимальный запас = Продажи/день × Дней до поставки × Коэфф.безопасности

Точка заказа = Продажи/день × Срок поставки + Страховой запас

Оборачиваемость (дни) = Средний остаток / Продажи в день
```

### Налоги (УСН)

```
УСН "Доходы" = Доход × 6% - Страховые взносы

УСН "Д-Р" = (Доход - Расход) × 15%

Доп. взнос 1% = (Доход - 300000) × 1%
```

## API Reference

### WB Statistics API

**Host:** `statistics-api.wildberries.ru`

**Endpoint:** `GET /api/v1/supplier/stocks`

**Описание:** Остатки на складах FBO

**Лимит:** 1 запрос/минуту

**Headers:**
```
Authorization: Bearer {WB_API_TOKEN}
```

---

### WB Prices API

**Host:** `discounts-prices-api.wildberries.ru`

**Endpoint:** `GET /api/v2/list/goods/filter`

**Описание:** Цены и скидки товаров

**Query params:**
| Параметр | Описание |
|----------|----------|
| limit | Кол-во записей (макс 1000) |
| offset | Смещение |
