## Структура карточки

### Обязательные элементы

| Элемент | Требование | Важность |
|---------|------------|----------|
| **Название** | До 200 символов | Критично |
| **Фото** | Минимум 1, рекомендуется 5-10 | Критично |
| **Описание** | До 6000 символов | Высокая |
| **Характеристики** | Все обязательные для категории | Критично |
| **Цена** | С учётом комиссий | Критично |
| **Категория** | Правильная категория | Критично |

### Дополнительные элементы

| Элемент | Описание | Влияние на конверсию |
|---------|----------|---------------------|
| **Видео** | До 10 минут | +15-25% |
| **Rich-контент** | Модули с фото/текстом | +20-30% |
| **360° фото** | Вращение товара | +10-15% |
| **Таблицы размеров** | Для одежды/обуви | +30-40% для одежды |

## Название товара

### Формула названия

```
[Бренд] [Тип товара] [Ключевые характеристики] [Для кого/чего]
```

**Примеры:**

**Хорошо:**
```
Samsung Galaxy S21 смартфон 128GB черный 5G
Adidas кроссовки мужские беговые черные размер 42
Philips утюг паровой 2400Вт керамическая подошва
```

**Плохо:**
```
Смартфон (слишком общее)
Супер-кроссовки ХИТ!!! (спам)
Утюг Philips модель ABC123XYZ (артикулы в названии)
```

### Правила названия

**✅ Делать:**
- Начинать с бренда
- Указывать ключевые характеристики
- Использовать понятные термины
- Писать для покупателя

**❌ Не делать:**
- Использовать CAPS LOCK
- Добавлять эмодзи
- Писать "ХИТ", "АКЦИЯ", "ЛУЧШИЙ"
- Повторять слова
- Использовать спецсимволы (!!! ### ***)

### Длина названия

| Длина | Рекомендация |
|-------|--------------|
| < 50 символов | Слишком короткое, мало информации |
| 50-120 символов | Оптимально |
| 120-200 символов | Допустимо |
| > 200 символов | Обрежется |

## Фотографии и видео

### Требования к фото

**Главное фото (1-е):**
- ✅ Товар на белом фоне
- ✅ Товар занимает 80-90% кадра
- ✅ Хорошее освещение
- ✅ Высокое разрешение (минимум 900×900px)
- ❌ Нет текста/логотипов/водяных знаков

**Дополнительные фото (2-10):**
- Разные ракурсы
- Детали товара
- Товар в использовании
- Размеры/габариты
- Комплектация

### Рекомендуемое количество фото

| Категория | Минимум | Оптимально |
|-----------|---------|------------|
| Одежда/обувь | 5 | 8-10 |
| Электроника | 3 | 5-7 |
| Товары для дома | 3 | 5-7 |
| Косметика | 2 | 4-6 |

### Инфографика

**Что показывать:**
- Преимущества товара
- Технические характеристики
- Размеры и габариты
- Комплектация
- Способы применения

**Формат:**
- Размер: 900×900px - 4000×4000px
- Формат: JPG, PNG
- Вес: до 10MB

### Видео

**Требования:**
- Длительность: 15 сек - 10 мин
- Формат: MP4, MOV, AVI
- Размер: до 500MB
- Разрешение: минимум 720p

**Что показывать в видео:**
- Распаковку товара
- Обзор со всех сторон
- Демонстрацию работы
- Сравнение с аналогами (если релевантно)

## Описание товара

### Структура описания

**1. Краткое описание (первые 100-150 символов):**
```
[Главное преимущество товара] + [Для кого/чего]
```

**Пример:**
```
Беспроводные наушники Samsung с активным шумоподавлением
для комфортного прослушивания музыки и звонков в любых условиях.
```

**2. Основная часть:**
- Преимущества (bullet points)
- Технические характеристики
- Область применения
- Комплектация
- Гарантия и сервис

**3. Призыв к действию:**
```
Закажите прямо сейчас и получите доставку за 1-2 дня!
```

### Правила описания

**✅ Делать:**
- Писать для покупателя (решение проблем)
- Использовать bullet points
- Указывать конкретные характеристики
- Отвечать на частые вопросы
- Структурировать текст

**❌ Не делать:**
- Копировать с других маркетплейсов
- Использовать спам-слова
- Писать сплошным текстом
- Обещать то, чего нет
- Сравнивать с конкурентами

### Длина описания

| Длина | Рекомендация |
|-------|--------------|
| < 500 символов | Мало информации |
| 500-2000 символов | Оптимально |
| 2000-6000 символов | Хорошо для сложных товаров |
| > 6000 символов | Максимум, обрежется |

## Rich-контент

### Что такое Rich-контент

Rich-контент — расширенное описание товара с визуальными модулями (фото, текст, таблицы).

**Преимущества:**
- Более детальная информация
- Лучшая конверсия (+20-30%)
- Профессиональный вид
- Выделение среди конкурентов

### Типы модулей

| Модуль | Описание | Использование |
|--------|----------|---------------|
| **Текст с фото** | Описание + изображение | Преимущества товара |
| **Галерея** | Несколько фото | Ракурсы, детали |
| **Таблица** | Структурированные данные | Характеристики, сравнения |
| **Видео** | Встроенное видео | Обзор, инструкция |
| **Баннер** | Широкое изображение | Акция, преимущество |

### Как создать Rich-контент

1. **В личном кабинете Ozon:**
   - Перейти в карточку товара
   - Раздел "Rich-контент"
   - Добавить модули

2. **Требования:**
   - Минимум 3 модуля
   - Максимум 15 модулей
   - Размер изображений: 900×600px - 4000×2000px

3. **Лучшие практики:**
   - Начинать с главного преимущества
   - Чередовать текст и изображения
   - Использовать таблицы для характеристик
   - Добавлять видео (если есть)

## Модерация карточек

### Сроки модерации

| Тип карточки | Срок |
|--------------|------|
| Новая карточка | 1-2 дня |
| Изменение карточки | 1 день |
| С нарушениями | Отклонение сразу |

### Частые причины отклонения

| Причина | Решение |
|---------|---------|
| Неправильная категория | Выбрать корректную |
| Плохие фото | Переделать по требованиям |
| Спам в названии | Убрать "ХИТ", "АКЦИЯ" |
| Неполные характеристики | Заполнить все обязательные |
| Несоответствие товара | Проверить соответствие |
| Дубль карточки | Удалить дубликат |

### Что проверяет модератор

✅ **Проверяется:**
- Соответствие фото товару
- Правильность категории
- Отсутствие запрещённых товаров
- Качество фото
- Заполнение обязательных полей
