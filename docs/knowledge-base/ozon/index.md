## Отличия от Wildberries

| Параметр | Ozon | Wildberries |
|----------|------|-------------|
| Модели работы | FBO, FBS, realFBS | FBO, FBS |
| Комиссии | Фиксированные по категориям | Плавающие |
| Rich-контент | Полноценный (модули) | Ограниченный |
| Аналитика | Детальная | Базовая |
| Выплаты | По графику | 2 раза в неделю |
| API | Ozon Seller API | WB Seller API |
