# Справочники атрибутов Ozon

Словари и атрибуты для категории **Ювелирные украшения / Браслеты** (category_id: 17027899, type_id: 87458883).

## Файлы

| Файл | Описание |
|------|----------|
| `ozon-attributes.json` | Обязательные атрибуты категории |
| `ozon-optional-attributes.json` | Опциональные атрибуты для контент-рейтинга |
| `ozon-dictionaries.json` | Справочные значения для атрибутов с dictionary_id |
| `ozon-dict-values.json` | Примеры значений словарей |

## Ключевые атрибуты для категории

## Использование

Справочники используются в:
- Skill `ozon-import-from-wb` — миграция товаров WB→Ozon
- MCP tool `ozon_get_category_attributes` — получение атрибутов
- Scripts `prepare-ozon-import.ts` — подготовка импорта

## Обновление справочников

```bash
# Получить актуальные атрибуты категории
npx tsx scripts/get-ozon-attributes.ts

# Получить значения словарей
npx tsx scripts/get-ozon-dict-values.ts
```
