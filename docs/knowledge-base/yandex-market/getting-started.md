## Как подключиться

### Требования к продавцу

| Тип продавца | Доступ | Особенности |
|--------------|--------|-------------|
| **ООО** | ✅ Полный | Все модели работы доступны |
| **ИП** | ✅ Полный | Все модели работы доступны |
| **Самозанятый** | ⚠️ Ограниченный | Только собственное производство |

### Основные шаги подключения

1. **Подача заявки**
   - Перейти на [partner.market.yandex.ru](https://partner.market.yandex.ru/)
   - Заполнить форму регистрации
   - Указать тип продавца (ООО/ИП/Самозанятый)
   - Предоставить контактные данные

2. **Проверка документов**
   - ИНН организации/ИП
   - ОГРН/ОГРНИП
   - Юридический/фактический адрес
   - Банковские реквизиты
   - Скан-копии уставных документов

3. **Заключение договора**
   - Оферта с Яндекс.Маркет
   - Электронная подпись (ЭЦП)
   - Согласие на обработку персональных данных

### Сроки подключения

| Этап | Срок |
|------|------|
| Проверка документов | 1-3 рабочих дня |
| Подписание договора | 1 день |
| Активация кабинета | Сразу после подписания |
| Первая выплата | Через 2-4 недели после первой продажи |

## Регистрация продавца

### Необходимые документы

**Для ООО:**
- Устав организации
- Свидетельство о регистрации (ОГРН)
- ИНН
- Реквизиты расчётного счёта
- Документы, подтверждающие полномочия директора

**Для ИП:**
- Свидетельство о регистрации ИП (ОГРНИП)
- ИНН
- Паспорт индивидуального предпринимателя
- Реквизиты расчётного счёта

**Для Самозанятых:**
- ИНН
- Паспорт
- Справка о постановке на учёт в качестве самозанятого
- Подтверждение собственного производства товаров

### Особенности для самозанятых

**Ограничения:**
- Только товары собственного производства
- Лимит выручки: 2.4 млн ₽/год
- Упрощённая отчётность
- Налог: 4% (физлица) или 6% (юрлица)

**Что можно продавать:**
- Товары ручной работы (handmade)
- Продукция собственного производства
- Авторские изделия

**Что нельзя:**
- Перепродажа готовых товаров
- Товары сторонних производителей

## Выбор модели работы

### Доступные модели

| Модель | Описание | Для кого |
|--------|----------|----------|
| **FBY** | Товар на складе Маркета | Масштабирование, стабильные продажи |
| **FBS** | Товар у продавца, доставка через Маркет | Тестирование, КГТ |
| **Экспресс** | Доставка 1-2 часа | Локальные продажи в крупных городах |
| **DBS** | Товар и доставка у продавца | Собственная логистика |

### Сравнение моделей

| Критерий | FBY | FBS | Экспресс | DBS |
|----------|-----|-----|----------|-----|
| **Хранение** | Маркет | Продавец | Продавец | Продавец |
| **Комплектация** | Маркет | Продавец | Продавец | Продавец |
| **Доставка** | Маркет | Маркет | Маркет (1-2ч) | Продавец |
| **Скорость** | 1-3 дня | 2-5 дней | 1-2 часа | Зависит от продавца |

**Рекомендации по выбору:**
- Начинающим → **FBS** (меньше рисков, тестирование)
- Для масштаба → **FBY** (приоритет в выдаче, быстрая доставка)
- Локально → **Экспресс** (сверхбыстрая доставка)
- Своя логистика → **DBS** (полный контроль)

## Первые шаги после регистрации

### 1. Настройка личного кабинета

**Обязательно:**
- ✅ Заполнить реквизиты для выплат
- ✅ Настроить способ доставки (FBY/FBS/DBS/Экспресс)
- ✅ Добавить банковские реквизиты
- ✅ Настроить уведомления (email, push)
- ✅ Настроить электронный документооборот (ЭДО)

**Рекомендуется:**
- Загрузить логотип магазина
- Заполнить описание магазина
- Настроить витрину магазина
- Подключить API (при необходимости)

### 2. Добавление товаров

**Минимальные требования к карточке:**
- Название товара
- Категория
- Описание
- Фотографии (минимум 1, рекомендуется 3-7)
- Цена
- Наличие (остатки)
- Обязательные характеристики для категории

**Рекомендации:**
- Заполнить все характеристики
- Добавить качественные фото (белый фон)
- Написать подробное описание
- Указать все доступные варианты (размеры, цвета)

### 3. Первая поставка (для FBY)

**Алгоритм:**
1. Создать поставку в личном кабинете
2. Подготовить товар (маркировка, упаковка)
3. Распечатать этикетки и документы
4. Выбрать склад Маркета
5. Доставить товар на склад
6. Дождаться приёмки (1-3 дня)

### 4. Первый заказ (для FBS/DBS/Экспресс)

**Подготовка:**
- Настроить график работы
- Подготовить упаковочные материалы
- Распечатать этикетки
- Настроить логистику (забор/доставка)

**Обработка:**
1. Получить уведомление о заказе
2. Собрать товар (FBS: до 24 часов)
3. Упаковать по требованиям
4. Передать курьеру или доставить самостоятельно

## Требования и ограничения

### Ограничения для новых продавцов

**Первые 2-4 недели:**
- Задержка выплат
- Обязательная модерация всех карточек
- Лимит на количество товаров
- Контроль качества работы

### Требования к товарам

**Обязательно:**
- Сертификаты соответствия (для определённых категорий)
- Маркировка «Честный ЗНАК» (где требуется)
- Декларация соответствия
- Инструкция на русском языке

**Категории с обязательной сертификацией:**
- Детские товары
- Электроника и бытовая техника
- Косметика и парфюмерия
- Продукты питания
- Медицинские изделия

### Запрещённые категории

**Не разрешены к продаже:**
- Оружие и боеприпасы
- Наркотики и психотропные вещества
- Контрафакт
- Алкоголь (требуется лицензия)
- Табачные изделия
- Лекарства (требуется лицензия)

## Комиссии и тарифы

### Базовые комиссии

| Категория | Комиссия Маркета |
|-----------|------------------|
| Электроника | 5-8% |
| Одежда и обувь | 10-15% |
| Товары для дома | 8-12% |
| Косметика | 10-15% |
| Продукты | 3-8% |

**Важно:** Актуальные комиссии для вашей категории смотрите в личном кабинете Маркета.

### Логистика (FBY)

**Хранение:**
- Первые 30 дней: от 3₽/литр/сутки
- 30-90 дней: от 5₽/литр/сутки
- Более 90 дней: от 10₽/литр/сутки

**Доставка покупателю:**
- Включена в комиссию Маркета

## Частые ошибки новичков

| Ошибка | Последствие | Решение |
|--------|-------------|---------|
| Плохие фото товаров | Низкая конверсия | Фото на белом фоне, минимум 3 шт |
| Неполное описание | Отказы, возвраты | Детальное описание всех характеристик |
| Неправильная категория | Блокировка карточки | Проверить категоризатор Маркета |
| Отсутствие сертификатов | Блокировка товара | Подготовить заранее для нужных категорий |
| Нет запаса на FBY | Out of stock | Поставлять с запасом на 30-60 дней |
