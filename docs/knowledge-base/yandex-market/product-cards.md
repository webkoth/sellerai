## Структура карточки

### Обязательные элементы

| Элемент | Требование | Важность |
|---------|------------|----------|
| **Название** | До 256 символов | Критично |
| **Категория** | Правильная категория | Критично |
| **Фото** | Минимум 1, рекомендуется 3-7 | Критично |
| **Описание** | До 6000 символов | Высокая |
| **Характеристики** | Все обязательные для категории | Критично |
| **Цена** | С учётом комиссий | Критично |
| **Остатки** | Актуальные остатки | Критично |

### Дополнительные элементы

- **Видео** — повышает конверсию на 20-30%
- **Rich-контент** — детальное описание с изображениями
- **Сертификаты** — для категорий с обязательной сертификацией
- **Инструкции** — руководства пользователя

## Создание карточки

### Способы создания

**1. Через личный кабинет:**
- Простой интерфейс
- Пошаговое заполнение
- Подходит для небольшого ассортимента

**2. Через Excel/CSV:**
- Массовая загрузка
- Шаблоны Маркета
- Для больших каталогов (> 50 товаров)

**3. Через API:**
- Автоматизация
- Интеграция с 1С/CRM
- Для профессионального использования

### Требования к фото

**Главное фото (1-е):**
- ✅ Товар на белом фоне
- ✅ Товар занимает 80-90% кадра
- ✅ Хорошее освещение
- ✅ Разрешение минимум 700×700px
- ❌ Нет текста/логотипов/водяных знаков

**Дополнительные фото:**
- Разные ракурсы товара
- Детали и особенности
- Товар в использовании
- Размеры/габариты
- Комплектация

**Рекомендуемое количество:**
- Минимум: 3 фото
- Оптимально: 5-7 фото
- Для одежды/обуви: 7-10 фото

## Оптимизация контента

### Название товара

**Формула:**
```
[Бренд] [Тип товара] [Ключевые характеристики] [Для кого/чего]
```

**Примеры:**

✅ **Хорошо:**
```
Apple iPhone 15 Pro смартфон 256GB титановый серый 5G
Nike кроссовки мужские Air Max 90 черные размер 43
Philips фен для волос 2200Вт керамическое покрытие
```

❌ **Плохо:**
```
Смартфон (слишком общее)
СУПЕР-КРОССОВКИ!!! ХИТ ПРОДАЖ (спам)
Фен Philips модель BHD176/00 (артикулы вместо характеристик)
```

### Описание товара

**Структура:**

1. **Краткое описание (первые 150 символов):**
   - Главное преимущество
   - Для кого товар

2. **Основная часть:**
   - Преимущества (bullet points)
   - Технические характеристики
   - Область применения
   - Комплектация
   - Гарантия

3. **Призыв к действию:**
   - Заказать с доставкой 1-3 дня
   - Гарантия возврата

**✅ Делать:**
- Писать для покупателя
- Использовать bullet points
- Указывать конкретные характеристики
- Отвечать на частые вопросы

**❌ Не делать:**
- Копировать с других маркетплейсов
- Использовать спам-слова ("ХИТ", "АКЦИЯ")
- Писать сплошным текстом
- Сравнивать с конкурентами

### Характеристики

**Обязательные (зависят от категории):**
- Бренд
- Модель
- Страна производства
- Гарантия
- Специфические для категории

**Рекомендуемые дополнительные:**
- Цвет (точное название)
- Материал
- Размеры и вес
- Особенности
- Сезон (для одежды)
- Возрастная группа

## Ранжирование в поиске

### Какие предложения попадают в результаты поиска

В поисковую выдачу попадает сразу несколько предложений — на первых местах продавцы, опережающие конкурентов по одному или нескольким из этих условий:

- высокое качество контента на карточке
- минимальная цена на Маркете
- самая быстрая доставка
- самая высокая ставка в бусте продаж
- больше всего продаж за последнее время
- лучшее предложение по совокупности факторов

### Как работает ранжирование в поиске

Когда все предложения, отвечающие на запрос покупателя отобраны, поисковый алгоритм выстраивает их в нужном порядке. То, насколько высоко будет ваше предложение в результатах поиска, зависит от нескольких факторов:

| Фактор | Вес | Описание |
|--------|-----|----------|
| **Релевантность** | 23,5% | Описание товара должно соответствовать запросу пользователя |
| **Персональные предпочтения** | 19,5% | Основаны на предыдущих покупках и поисковых запросах пользователя |
| **Цена товара** | 18% | Чем ближе к минимальной на рынке, тем лучше |
| **Популярность товара** | 15,5% | Как часто товар заказывают, добавляют в корзину и избранное, просматривают карточку |
| **Качество карточки** | 14,5% | Чем ближе к максимальным 100 баллам, тем лучше |
| **Срок доставки и рейтинг** | 9% | Скорость доставки и рейтинг магазина |

### Как поднять предложение выше

**1. Подробно заполняйте карточки**

Чем больше подробной и качественной информации о товаре на карточке — тем выше место товара в поиске.

> **Пример:** Покупатель ищет телевизор с диагональю 32 дюйма. Подходящий телевизор не попадет ему на глаза, если в карточке товара диагональ не указана.

**2. Устанавливайте привлекательные цены**

- Проверьте конкурентность цен — сравнение с ценами других продавцов
- Предложения с лучшей ценой на рынке поднимаются на более высокие позиции
- Скидки и промокоды тоже сделают цену привлекательнее

**3. Улучшайте условия доставки**

- Настройте продажи со склада Маркета по модели FBY
- Храните товары ближе к покупателям — в регионах со спросом
- Для FBS: включите опцию "Хочу быстрее попадать в доставку"
- Добавьте выходные в график работы для ускорения отгрузок
- Подключите Экспресс для доставки за пару часов

**4. Собирайте отзывы о товарах**

Отзывы влияют на позицию карточки в результатах поиска. Мотивируйте покупателей оставлять отзывы, предложив вознаграждение баллами Яндекс Плюса.

**5. Проверяйте рейтинг магазина**

Высокий рейтинг магазина привлекает покупателей и влияет на место в поиске.

> **Источник:** [Официальная документация Яндекс.Маркета](https://yandex.ru/support/marketplace/ru/tutorials/search)

## Управление остатками и ценами

### Остатки

**Синхронизация:**
- Автоматическая (через API)
- Ручная (через кабинет)
- Excel/CSV загрузка

**Важно:**
- Обновлять остатки ежедневно
- Не допускать out of stock
- Для FBY: контролировать запас на складе

### Цены

**Управление ценами:**
- Базовая цена
- Акционная цена
- Цена со скидкой
- Промокоды

**Автоматизация:**
- Можно настроить правила изменения цен
- Интеграция с системами учёта
- Мониторинг цен конкурентов

**Рекомендации:**
- Конкурентная цена = больше продаж
- Учитывать комиссии Маркета
- Участие в акциях для роста продаж

## Модерация

### Сроки модерации

| Тип | Срок |
|-----|------|
| Новая карточка | 1-2 дня |
| Изменение карточки | 1 день |
| С нарушениями | Отклонение сразу |

### Частые причины отклонения

| Причина | Решение |
|---------|---------|
| Неправильная категория | Выбрать корректную категорию |
| Плохие фото | Переделать по требованиям |
| Спам в названии/описании | Убрать "ХИТ", "АКЦИЯ", эмодзи |
| Неполные характеристики | Заполнить все обязательные поля |
| Отсутствие сертификатов | Загрузить сертификаты (где требуется) |
| Дубль карточки | Удалить дубликат |

### Что проверяет модератор

✅ **Проверяется:**
- Соответствие фото товару
- Правильность категории
- Отсутствие запрещённых товаров
- Качество фото (разрешение, фон)
- Заполнение обязательных полей
- Отсутствие спама в тексте
