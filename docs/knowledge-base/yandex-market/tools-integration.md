# Инструменты и интеграция на Яндекс.Маркет

> **Релевантно для skills:** yandex-market-basics

## Приложение Яндекс Маркет для продавцов

### Мобильное приложение

**Возможности:**
- Просмотр заказов
- Обработка заказов (подтверждение, отмена)
- Уведомления в реальном времени
- Статистика продаж
- Общение с покупателями

**Платформы:**
- iOS (App Store)
- Android (Google Play)

**Преимущества:**
- Работа на ходу
- Быстрая реакция на заказы
- Push-уведомления

## API и модули

### Yandex Market Partner API

**Официальное API для автоматизации:**
- [Документация API](https://yandex.ru/dev/market/partner-api/doc/ru/)

**Основные возможности:**
- Управление товарами (добавление, редактирование, удаление)
- Управление остатками и ценами
- Обработка заказов
- Получение отчётов
- Управление поставками (FBY)

### Интеграции

**С популярными системами:**
- 1С: Управление торговлей
- 1С: Розница
- CRM-системы
- Складские программы
- Маркетплейс-агрегаторы

**Готовые модули:**
- Модули для CMS (битрикс, opencart и др.)
- Плагины для учётных систем
- Коннекторы к агрегаторам

## Помощь агентства

### Сертифицированные агентства

**Маркет рекомендует проверенные агентства:**

**Услуги агентств:**
- Подключение к Маркету
- Создание и оптимизация карточек
- Настройка рекламы
- Управление продажами
- Аналитика и отчёты

**Когда стоит обратиться:**
- Нет опыта работы на маркетплейсах
- Большой каталог товаров
- Нужна профессиональная настройка рекламы
- Требуется регулярное управление

### Поиск агентства

**Критерии выбора:**
- Опыт работы с Маркетом
- Кейсы в вашей категории
- Отзывы других продавцов
- Прозрачность условий
- Адекватная стоимость услуг

## Кредиты для продавцов

### Кредитование

**Маркет предлагает:**
- Партнёрство с банками
- Льготные условия для продавцов
- Кредиты на развитие бизнеса

**Цели кредита:**
- Закуп товара
- Расширение ассортимента
- Оборотные средства
- Маркетинг и реклама

### Условия

**Как получить:**
1. Подать заявку через кабинет
2. Предоставить документы
3. Дождаться решения банка
4. Получить средства
