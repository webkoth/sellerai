## Отличия от других маркетплейсов

| Параметр | Яндекс.Маркет | Wildberries | Ozon |
|----------|---------------|-------------|------|
| Модели работы | FBY, FBS, DBS, Экспресс | FBO, FBS | FBO, FBS, realFBS |
| Экспресс-доставка | Да (1-2 часа) | Нет | Нет |
| Контент-доска | Да | Нет | Частично |
| Аналитика | Детальная | Базовая | Детальная |
| API | Yandex Market Partner API | WB Seller API | Ozon Seller API |
