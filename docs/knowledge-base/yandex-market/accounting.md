# Расчёты и бухгалтерия на Яндекс.Маркет

> **Категория:** Выплаты, договор, документы
> **Релевантно для skills:** yandex-market-basics, mp-accounting
> **Обновлено:** 2025-12-29

## Договор с Маркетом

### Тип договора

**Агентский договор:**
- Маркет — агент
- Продавец — принципал
- Маркет продаёт товары от имени продавца

**Особенности:**
- Отчёт агента (комиссионера)
- НДС по агентским услугам
- Первичные документы от Маркета

## Выплаты за товары

### График выплат

**Стандартный график:**
- Выплаты 2 раза в месяц
- 5-го и 20-го числа
- За предыдущий период

**Первая выплата:**
- Через 2-4 недели после первой продажи
- Может быть задержка для проверки

### Схема выплат

**Расчёт:**
```
Выручка от продаж
- Комиссия Маркета (3-15%)
- Логистика (для FBY/FBS)
- Хранение (для FBY)
- Возвраты
= Сумма к выплате
```

## Электронный документооборот (ЭДО)

### Настройка ЭДО

**Операторы ЭДО:**
- СБИС
- Контур.Диадок
- Такском
- Другие операторы

**Что нужно:**
1. Подключить оператора ЭДО
2. Указать данные в кабинете Маркета
3. Подписать документы электронной подписью (ЭЦП)

### Документы

**Маркет отправляет:**
- Отчёт агента (комиссионера)
- Счета-фактуры
- Акты сверки
- УПД (универсальный передаточный документ)

## Бухгалтерская отчётность

### Отчёты в кабинете

**Доступные отчёты:**
- Продажи и выплаты
- Комиссии и услуги
- Возвраты
- Акты сверки
- Детализация операций

### Работа с 1С

**Интеграция:**
- Автоматическая выгрузка данных
- Импорт в 1С: Бухгалтерия
- Формирование проводок
- Отчёты для налоговой

## Налогообложение

### УСН (упрощённая система)

**УСН Доходы (6%):**
- Налог с выручки 6%
- Проще учёт
- Для небольших объёмов

**УСН Доходы минус расходы (15%):**
- Налог с прибыли 15%
- Можно учитывать расходы
- Для больших объёмов с высокими затратами

### Самозанятые

**Налог:**
- 4% при продаже физлицам
- 6% при продаже юрлицам
- Лимит: 2.4 млн ₽/год
