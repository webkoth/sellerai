# Аналитика на Яндекс.Маркет

> **Категория:** Метрики, отчёты, анализ
> **Релевантно для skills:** yandex-market-basics
> **Обновлено:** 2025-12-29

## Ключевые показатели

### Основные метрики

| Метрика | Описание | Норма |
|---------|----------|-------|
| **Показы** | Сколько раз карточку видели | Чем больше, тем лучше |
| **Клики** | Переходы в карточку | > 1% от показов |
| **CTR** | Click-Through Rate (показы → клики) | 1-3% |
| **Конверсия в корзину** | Клики → добавления в корзину | 10-20% |
| **Конверсия в заказ** | Корзина → заказы | 30-50% |
| **Выручка** | Общие продажи | Рост месяц к месяцу |

## Ваша аналитика

### Аналитика продаж

**Доступные отчёты:**
- Продажи по товарам
- Продажи по периодам
- Динамика роста
- ABC-анализ товаров

### Конкурентная позиция

**Что показывает:**
- Ваше место в категории
- Сравнение с конкурентами
- Доля рынка
- Тренды категории

### География продаж

**Отчёты:**
- Продажи по регионам
- Популярные города
- Охват доставки
- Региональная динамика

### Аналитика брендов

**Для брендов:**
- Продажи всех товаров бренда
- Популярные SKU
- Рост узнаваемости
- Сравнение с конкурентами
