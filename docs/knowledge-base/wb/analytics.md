## Аналитика продавца

### Раздел «Аналитика продавца»
- Централизованный доступ к метрикам
- Дашборды с ключевыми показателями
- Настраиваемые отчёты

### Сводка аналитики
**Основные метрики:**
- Продажи за период
- Выручка
- Средний чек
- Количество заказов
- Динамика по дням

### Отчёт «Воронка продаж. Сравнение с периодом»

**Этапы воронки:**
1. **Просмотры** - сколько раз товар был показан
2. **Переходы в карточку** - клики на товар
3. **Добавления в корзину** - товар добавлен в корзину
4. **Заказы** - оформленные заказы
5. **Выкупы** - фактически выкупленные товары

**Конверсии:**
- Просмотры → Корзина (CTR)
- Корзина → Заказ (CR1)
- Заказ → Выкуп (CR2)

**Сравнение:**
- Текущий период vs предыдущий
- Выявление трендов
- Оценка эффективности изменений

### Отчёт «Воронка продаж. Сравнение с площадкой»
- Ваши показатели vs средние по WB
- Бенчмаркинг конверсий
- Выявление точек роста

### Отчёт «Портрет покупателя»

**Демография:**
- Пол

**Поведение:**

**Применение:**
- Таргетинг рекламы

### Отчёт «Календарь событий»

### Отчёт «История остатков»

### Отчёт «Поставки по регионам»

## Частые ошибки

### Ошибка 1: Игнорирование аналитики
- **Последствие:** Упущенные возможности, неэффективные решения
- **Решение:** Регулярный анализ ключевых метрик

### Ошибка 2: Фокус только на продажах
- **Последствие:** Не видны проблемы в воронке
- **Решение:** Анализировать всю воронку от показов до выкупа

### Ошибка 3: Нет сравнения с конкурентами
- **Последствие:** Непонимание своего положения на рынке
- **Решение:** Использовать отчёты сравнения карточек и ниш

### Ошибка 4: Не использование поисковой аналитики
- **Последствие:** Плохая видимость товаров в поиске
- **Решение:** Оптимизация под реальные поисковые запросы
