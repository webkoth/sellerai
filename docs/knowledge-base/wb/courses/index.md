# Курсы WB Pro

> **Источник:** https://pro.wildberries.ru/courses
> **Формат:** Видеокурсы (бесплатные)
> **Обновлено:** 2025-12-29

## Описание

Курсы WB Pro — это бесплатные обучающие программы от Wildberries для продавцов, менеджеров и партнёров. Все курсы содержат видеоуроки, практические задания и доступны 24/7.

**Особенности:**
- Создаются экспертами Wildberries и успешными продавцами
- Включают пошаговые инструкции и практические кейсы
- Подходят как для начинающих, так и для опытных продавцов
- Бонусы: скидки на продукты WB, баллы на продвижение

**Статистика:** Более 100 000 продавцов прошли курсы WB Pro

## Как использовать курсы

**Для начинающих:**
1. Начните с курса «Бизнес на Wildberries: как начать с нуля»
2. Пройдите «Работа с карточкой товара»
3. Изучите «Модели продаж»
4. Освойте «Финансовые инструменты»

**Для опытных продавцов:**
- «Всё о продвижении товаров» — углубленная реклама
- «Доставка силами продавца» — расширение моделей продаж
- «Продвижение через контент: Wibes» — новые каналы

**Для менеджеров:**
- «Сертификация менеджеров маркетплейсов» — профессиональная квалификация
