## Баланс магазина

### Как работать с балансом

**Структура баланса:**
- **Доступно к выводу** - средства, готовые к выплате
- **В пути** - деньги за товары в обработке
- **Заблокировано** - средства под штрафы/удержания

**Операции:**
- Начисления за продажи
- Удержания и комиссии
- Логистические расходы
- Штрафы
- Хранение

### Как вывести деньги на свой счёт

**Стандартный вывод:**
- График выплат: раз в неделю (понедельник)
- Автоматический перевод на расчётный счёт
- Срок зачисления: 1-3 рабочих дня

**Настройка вывода:**
1. Указать расчётный счёт
2. Подтвердить реквизиты
3. Дождаться даты выплаты

### Услуга «Вывести сейчас»

**Что это:**
- Досрочный вывод средств
- Доступно в любой день
- За определённую комиссию

**Условия:**
- Комиссия ~1-2%
- Минимальная сумма
- Зачисление в течение 1-2 дней

**Когда использовать:**
- Срочная потребность в средствах
- Пополнение оборотных средств
- Оплата поставщиков

### Как оплатить долг

**Причины задолженности:**
- Возвраты товаров
- Штрафы и удержания
- Расходы превысили доход

**Способы погашения:**
- Пополнение баланса
- Удержание из будущих продаж
- Банковский перевод

## Финансовые отчёты

### Еженедельные отчёты реализации

**Что включает:**
- Продажи за неделю
- Возвраты
- Комиссии WB
- Логистические расходы
- Итоговая выплата

**Формат:**
- Универсальный передаточный документ (УПД)
- Акт об оказанных услугах
- Реализация товара

**Получение:**
- Автоматически каждый понедельник
- Доступ в разделе «Документы»
- Электронный документооборот (ЭДО)

### Детализация еженедельного отчёта реализации

**Разбивка по товарам:**
- Количество проданных единиц
- Стоимость продаж
- Комиссия за каждый товар
- Логистика по товару
- Чистая прибыль

**Разбивка по операциям:**
- Продажи
- Возвраты от покупателей
- Компенсации
- Корректировки

## Документы

### Электронный документооборот c Wildberries

**Поддерживаемые операторы ЭДО:**
- Контур.Диадок
- СБИС
- 1С-ЭДО
- Такском
- Тензор

**Типы документов в ЭДО:**
- УПД (Универсальный передаточный документ)
- Акты оказания услуг
- Счета-фактуры
- Корректировочные УПД

**Настройка:**
1. Выбрать оператора ЭДО
2. Подключить интеграцию
3. Настроить автоматическую подпись
4. Получать документы в электронном виде

### УПД на уведомление о выкупе

**Что это:**
- Документ по проданным товарам
- Основание для налогового учёта
- Формируется еженедельно

**Содержание:**
- Список проданных товаров
- Суммы реализации
- Комиссии и удержания

### Формирование УПД «ДОП» на товары с обязательной маркировкой

**Для каких товаров:**
- Товары с КИЗ (обувь, одежда и др.)
- Поставки типа «Короб»

**Особенности:**
- Указание кодов маркировки
- Вывод КИЗ из оборота
- Передача данных в ГИС МТ (Честный ЗНАК)

### Передача кодов маркировки

**Поставки «Монопаллета»:**
- Передача КИЗ при отгрузке
- Автоматический вывод из оборота
- Синхронизация с Честным ЗНАКом

### Акт корректировки

**Когда формируется:**
- Расхождения при приёмке
- Недостачи
- Излишки
- Брак

**Применение:**
- Корректировка первичных документов
- Налоговый учёт
- Урегулирование споров

### Документы по движению товара

**Типы документов:**
- Поставки на склад
- Перемещения между складами
- Возвраты продавцу
- Списания

### Как сформировать чеки: инструкция для самозанятых

**Требования:**
- Статус самозанятого
- Регистрация в «Мой налог»
- Чеки по каждой продаже

**Процесс:**
1. Получить данные о продажах из отчёта WB
2. Сформировать чеки в приложении «Мой налог»
3. Отправить чеки покупателям (при необходимости)

## Типичные ошибки

### Ошибка 1: Неправильный расчёт доступного баланса
- **Последствие:** Планирование расходов на заблокированные средства
- **Решение:** Учитывать только «доступно к выводу»

### Ошибка 2: Игнорирование УПД
- **Последствие:** Проблемы с налоговой
- **Решение:** Своевременно подписывать все документы в ЭДО

### Ошибка 3: Не контролировать удержания
- **Последствие:** Неожиданные списания
- **Решение:** Регулярно проверять отчёт по удержаниям

### Ошибка 4: Не использовать финансовые инструменты
- **Последствие:** Нехватка оборотных средств
- **Решение:** Рассмотреть займы и овердрафты для роста
