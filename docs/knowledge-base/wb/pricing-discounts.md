## Управление ценами

### Как устанавливать цены и скидки

**Основные принципы:**
- Цена указывается **до скидки**
- Скидка устанавливается в % или фиксированной суммой
- Финальная цена = Цена - Скидка - Промокод

**Окно управления ценой товара:**
- Единый интерфейс для управления ценой
- История изменений цены
- Прогноз влияния на продажи

### Индекс цен

**Что это:**
- Показатель конкурентоспособности цены товара
- Сравнение с аналогами на площадке

**Как использовать:**
- Зелёный индекс - цена ниже конкурентов
- Жёлтый - средняя цена
- Красный - цена выше конкурентов

### Карантин цен

**Суть механизма:**
- Защита от резких скачков цен
- Ограничение частоты изменения цены

**Правила:**
- После изменения цены - карантин 3 дня
- В течение карантина цену менять нельзя
- Исключения: снижение цены, участие в акциях

### Как проверить обновление цен и скидок
- Время обновления: до 15 минут
- Проверка в карточке товара
- API для автоматизации

## Скидки и акции

### Скидка для WB Клуба

**Что это:**
- Специальная скидка для участников WB Клуба
- Устанавливается продавцом дополнительно к основной

**Выгоды:**
- Повышение видимости товара
- Приоритет в выдаче для клубников
- Рост конверсии

### Минимальная цена и блокировка в автоакциях

**Минимальная цена:**
- Защита от чрезмерных скидок
- Установка "пола" цены

**Блокировка в автоакциях:**
- Запрет участия товара в автоакциях
- Контроль маржинальности

### Софинансирование рассрочки продавцом

**Механизм:**
- Продавец частично компенсирует процент по рассрочке
- Привлекает покупателей с ограниченным бюджетом

**Настройка:**
- Выбор процента софинансирования (0-100%)
- Расчёт влияния на маржу

### Кешбэк на товары

**Как установить:**
- Выбор % кешбэка (до 20%)
- Начисление баллов WB Клуба покупателю

**Эффект:**
- Повышение привлекательности товара
- Стимулирование повторных покупок

## Инструменты ценообразования

### Стратегии ценообразования

**1. Конкурентное ценообразование:**
- Ориентация на цены конкурентов
- Использование индекса цен

**2. Ценообразование на основе маржи:**
- Расчёт от себестоимости
- Целевая маржа 30-50%

**3. Динамическое ценообразование:**
- Изменение цены в зависимости от спроса
- Автоматизация через API

**4. Психологическое ценообразование:**
- Цены 999₽ вместо 1000₽
- Округление для премиум-товаров

### Калькулятор прибыли

**Расчёт:**
- Цена товара
- Комиссия WB (по категории)
- Логистика и хранение
- Возвраты (~10-15%)
- **= Чистая прибыль**

### Анализ эластичности цены
- Влияние изменения цены на объём продаж
- Поиск оптимальной точки цена/объём

## Работа с ценами по моделям продаж

### FBW (Склад WB)
- Учёт стоимости хранения
- Платное хранение при низкой оборачиваемости

### FBS (Маркетплейс)
- Ниже комиссия = больше маневра для цены
- Контроль логистических затрат

### DBS/EDBS (Витрина)
- Самая низкая комиссия
- Свободное ценообразование

## Частые ошибки

### Ошибка 1: Слишком частое изменение цен
- **Последствие:** Карантин цен, потеря позиций
- **Решение:** Планировать изменения заранее

### Ошибка 2: Игнорирование индекса цен
- **Последствие:** Неконкурентная цена
- **Решение:** Мониторить конкурентов

### Ошибка 3: Чрезмерные скидки
- **Последствие:** Низкая маржа, убытки
- **Решение:** Установить минимальную цену

### Ошибка 4: Не учитывать все затраты
- **Последствие:** Продажа в минус
- **Решение:** Использовать калькулятор прибыли
