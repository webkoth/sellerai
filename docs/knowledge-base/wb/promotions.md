## Акции на Wildberries

### Типы акций:

**1. Акции от WB:**
- **Программные акции** (Черная пятница, НГ и т.д.)
- **Категорийные акции** (Красота, Электроника и т.д.)
- **Региональные акции**

**2. Автоакции:**
- **Автоматическое участие** при соблюдении условий
- **Гибкая настройка** участия

**3. Индивидуальные акции:**
- **СПП** - скидка постоянного покупателя
- **Кешбэк**
- **Промокоды**

### Ранжирование товаров в акциях

**Факторы ранжирования:**
1. **Глубина скидки** - чем больше, тем выше
2. **Рейтинг товара** - отзывы и оценки
3. **Продажи** - история продаж
4. **Остатки** - наличие товара
5. **Цена** - конкурентоспособность

**Как попасть в топ акции:**
- Установить скидку >= среднего по акции
- Иметь высокий рейтинг (4.5+)
- Обеспечить наличие товара

## Автоакции

### Как работают автоакции

**Механизм:**
- WB автоматически добавляет товар в акцию
- Если соблюдены условия (цена, рейтинг, остатки)
- Скидка устанавливается автоматически

**Настройка автоакций:**
- **Минимальная цена** - защита от чрезмерных скидок
- **Блокировка** - запрет участия в автоакциях
- **Максимальная скидка** - лимит на скидку

### Как добавить в автоакцию все товары одной кнопкой
- Массовое добавление через ЛК
- Фильтры по категориям и характеристикам
- Единые настройки для группы товаров

### Глубина скидки в автоакциях

**Рекомендации:**
- Минимум 20% для участия
- Оптимум 30-40% для топовых позиций
- Максимум - по вашей маржинальности

**Расчёт оптимальной скидки:**
```
Оптимальная скидка = Средняя скидка в акции + 5-10%
```

## СПП (Скидка постоянного покупателя)

### Что это:
- Автоматическая скидка для покупателей WB Клуба
- Устанавливается продавцом дополнительно

### Как работает:

**Механизм:**
1. Покупатель состоит в WB Клубе
2. Продавец установил СПП на товар
3. При покупке применяется СПП автоматически

**Размер СПП:**
- Минимум: 1%
- Максимум: без ограничений (на усмотрение продавца)
- Рекомендуемый: 5-10%

### Выгоды СПП:

**Для продавца:**
- Повышение конверсии у членов клуба
- Приоритет в выдаче для клубников
- Лояльность покупателей

**Для покупателя:**
- Дополнительная скидка
- Накопление баллов
- Эксклюзивные предложения

## Стратегии участия в акциях

### Стратегия 1: Агрессивная
- **Цель:** Максимизация продаж
- **Скидка:** 40-60%
- **Подходит для:** Распродажи остатков, увеличения оборачиваемости

### Стратегия 2: Сбалансированная
- **Цель:** Рост продаж при сохранении маржи
- **Скидка:** 25-35%
- **Подходит для:** Регулярного участия в акциях

### Стратегия 3: Консервативная
- **Цель:** Поддержание позиций без потери маржи
- **Скидка:** 15-20%
- **Подходит для:** Премиум-товаров, уникальных позиций

## Расчёт эффективности участия в акции

### Метрики для оценки:

**1. Прирост продаж:**
```
Прирост = (Продажи в акцию - Средние продажи) / Средние продажи * 100%
```

**2. Сохранение маржи:**
```
Маржа в акцию = Цена со скидкой - Все затраты
```

**3. ROI акции:**
```
ROI = (Дополнительная прибыль - Затраты на скидку) / Затраты на скидку * 100%
```

**Целевые показатели:**
- Прирост продаж: +200-500%
- Маржа: сохранить >20%
- ROI: >0%

## Частые ошибки

### Ошибка 1: Нет остатков во время акции
- **Последствие:** Потеря позиций, штрафы
- **Решение:** Планировать поставки заранее

### Ошибка 2: Слишком маленькая скидка
- **Последствие:** Низкие позиции в акции
- **Решение:** Анализировать средние скидки

### Ошибка 3: Участие во всех акциях
- **Последствие:** Снижение маржи, восприятие "дешевого" товара
- **Решение:** Выборочное участие в ключевых акциях
