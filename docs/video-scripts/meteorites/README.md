## Единая система оверлеев (для следующего шага)

Все оверлеи во всех сценариях размечены **4 ролями**. На следующем шаге зададим для каждой роли единый визуальный стиль (шрифт, цвет, размер, позиция, анимация) — один раз, и применим ко всем 6 роликам.

| Роль | Назначение | Появляется |
|------|------------|------------|
| **TITLE** | тема ролика, крупно | на хуке (0–3 сек) |
| **FACT** | цифры и термины (4,5 млрд лет, Fe+Ni, узор Видманштеттена) | в фактических блоках |
| **KEY** | ключевая мысль / акцент | в смысловых блоках |
| **CTA** | призыв к действию | в финале |

### Что решаем при сборке стиля

- [ ] Шрифт (заголовочный для TITLE/CTA + текстовый для FACT/KEY).
- [ ] Палитра: основной цвет, акцент для FACT, фон-плашка/обводка для читаемости на тёмном.
- [ ] Позиция и безопасные поля (нижняя треть vs центр, отступ под интерфейс Reels/Shorts).
- [ ] Анимация появления/ухода (fade / slide / typewriter) — единая для всех ролей.
- [ ] Локализация под площадки: Reels, Shorts, VK Clips, TikTok (разные «слепые зоны» интерфейса).

## Перед съёмкой — общий чек-лист

- [ ] В ролике 4 заполнены все `[плейсхолдеры]` данными карточки.
- [ ] Факты не выдуманы (вес, размер, тип, комплектация — из карточки).
- [ ] Нет обещаний лечения, защиты, денег, сверхспособностей.
- [ ] Хук в первые 1–3 сек, без приветствия.
- [ ] Один ролик — одна мысль.
- [ ] CTA мягкий и обращён к зрителю.
