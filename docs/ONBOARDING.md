## День 1: Подключение (2-3 часа)

### 1.1 Получение доступов

- [ ] **Wildberries API-токен**
  - ЛК: seller.wildberries.ru → Настройки → Доступ к API
  - Нужные права: Контент, Цены, Статистика, Продвижение, Отзывы
  - Записать в `.env`: `WB_API_TOKEN="..."`

- [ ] **Ozon API-ключи** (если подключаем)
  - ЛК: seller.ozon.ru → Настройки → API-ключи
  - Client ID + API Key
  - Записать в `.env`: `OZON_CLIENT_ID="..."`, `OZON_API_TOKEN="..."`

- [ ] **Яндекс.Маркет токен** (если подключаем)
  - ЛК: partner.market.yandex.ru → Настройки → Токены
  - OAuth токен + Business ID + Campaign ID
  - Записать в `.env`: `YM_TOKEN="..."`, `YM_BUSINESS_ID="..."`, `YM_CAMPAIGN_ID="..."`

### 1.2 Настройка workspace

- [ ] Скопировать workspace SellerAI для клиента
  ```bash
  cp -r /Users/minas/projects/sellerai /Users/minas/projects/clients/[имя_клиента]
  ```

- [ ] Обновить `.env` файл с токенами клиента

- [ ] Проверить подключение к API
  ```
  # Wildberries
  wb_get_seller_info → должен вернуть название магазина

  # Ozon
  ozon_get_products → должен вернуть список товаров

  # Яндекс.Маркет
  ym_get_campaigns → должен вернуть список магазинов
  ```

### 1.3 Первичная синхронизация

- [ ] Загрузить остатки: `wb_get_inventory`
- [ ] Загрузить карточки: `wb_get_cards`
- [ ] Загрузить цены: `wb_get_prices`
- [ ] Загрузить заказы (30 дней): `wb_get_orders`
- [ ] Загрузить отзывы: `wb_get_reviews`
- [ ] Загрузить рекламные кампании: `wb_get_campaigns`

**Статус День 1:** [ ] Завершён

## День 2-3: Аудит магазина (4-6 часов)

### 2.1 ABC-анализ ассортимента

- [ ] Запустить `/abc-analysis`
- [ ] Выписать товары категории A (80% выручки)
- [ ] Выписать товары категории C (кандидаты на вывод)
- [ ] Проверить товары с нулевыми продажами

**Результат:** Топ-10 товаров по выручке, проблемные товары

### 2.2 Анализ воронки продаж

- [ ] Запустить `/funnel` для топ-10 товаров
- [ ] Проверить конверсии на каждом этапе:
  - Показы → Клики (CTR, норма 3-7%)
  - Клики → Корзина (норма 10-15%)
  - Корзина → Заказ (норма 30-50%)
  - Заказ → Выкуп (норма 70-85%)
- [ ] Выявить узкие места в воронке

**Результат:** Точки потерь в воронке, рекомендации

### 2.3 SEO-аудит карточек

- [ ] Запустить `/seo-audit` для топ-10 товаров
- [ ] Проверить:
  - Название (ключевые слова, длина)
  - Описание (структура, ключи)
  - Характеристики (заполненность)
  - Фото (количество, качество)

**Результат:** Список карточек с SEO-проблемами

### 2.4 Аудит рекламы

- [ ] Запустить `/ad-analysis`
- [ ] Проверить:
  - ROI каждой кампании (норма > 100%)
  - CTR (норма 3-7%)
  - Неэффективные ключевые слова
  - Расход vs выручка

**Результат:** Кампании для оптимизации, рекомендации по бюджету

### 2.5 Юнит-экономика

- [ ] Запустить `/unit-economics` для топ-5 товаров
- [ ] Проверить маржинальность (норма 25-40%)
- [ ] Выявить убыточные товары
- [ ] Рассчитать точку безубыточности

**Результат:** Маржа по ключевым товарам, проблемные позиции

**Статус День 2-3:** [ ] Завершён
