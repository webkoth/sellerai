## МЕТОДОЛОГИЯ РАСЧЁТА

### Формулы

```
Розничная цена (оценка) = К перечислению / 0.82
    (WB удерживает ~18% от розничной цены: комиссия + логистика)

Комиссия WB = Розничная цена - К перечислению

Себестоимость партии = Продано шт × Себестоимость единицы

Валовая прибыль = К перечислению - Себестоимость партии

Маржа = Валовая прибыль / К перечислению × 100%

ROI = Валовая прибыль / Себестоимость × 100%
```

### Источники данных

| Показатель | Источник |
|------------|----------|
| К перечислению | WB API (отчёт о реализации) |
| Продано штук | WB API (выкупленные товары) |
| Себестоимость | Данные клиента (CSV) |
| Комиссия WB | Расчётная (18% от розницы) |
