# Wildberries — Формулы метрик

> Единый источник формул для расчёта метрик Wildberries
> Используется всеми Skills, Commands и Agents

---

## 📊 Финансовые метрики

### Маржинальность

**Формула:**
```
Маржинальность = (Выручка - Все расходы) / Выручка × 100%
```

**Расшифровка:**
- **Выручка** = Цена продажи × Количество проданных единиц
- **Все расходы** = Себестоимость + Комиссия WB + Логистика + Хранение + Реклама + Налог

**Бенчмарки:**
- **< 15%** — критично (товар убыточный или близок к этому)
- **15-25%** — нормально (допустимая маржа для работы)
- **25-40%** — хорошо (здоровая маржа)
- **> 40%** — отлично (высокомаржинальный товар)

**Пример:**
- Выручка: 10,000₽
- Расходы: 6,500₽
- Маржинальность = (10,000 - 6,500) / 10,000 × 100% = **35%** ✅ Хорошо

---

### Рентабельность вложений (ROE)

**Формула:**
```
Рентабельность = Прибыль / Себестоимость × 100%
```

**Расшифровка:**
- **Прибыль** = Выручка - Все расходы
- **Себестоимость** = Закупочная цена + Доставка до склада

**Интерпретация:**
- Показывает сколько рублей прибыли вы получаете с каждого вложенного рубля
- **40%** = с каждых 100₽ вложений получаете 40₽ прибыли

**Пример:**
- Себестоимость: 500₽
- Прибыль: 200₽
- Рентабельность = 200 / 500 × 100% = **40%**

---

### ROI рекламы

**Формула:**
```
ROI = (Доход от рекламы - Расход на рекламу) / Расход на рекламу × 100%
```

**Альтернативная метрика - ROAS (Return on Ad Spend):**
```
ROAS = Доход от рекламы / Расход на рекламу
```

**Связь:** ROI = (ROAS - 1) × 100%

**Бенчмарки ROI:**
- **< 0%** — убыточная реклама (выключить)
- **0-100%** — окупается частично (оптимизировать)
- **100-200%** — окупается полностью (хорошо)
- **> 200%** — высокоэффективная реклама (масштабировать)

**Бенчмарки ROAS:**
- **< 1.0** — убыточная
- **1.0-2.0** — окупается частично
- **2.0-3.0** — окупается полностью
- **> 3.0** — высокоэффективная

**Пример:**
- Расход на рекламу: 10,000₽
- Доход от рекламы: 30,000₽
- ROI = (30,000 - 10,000) / 10,000 × 100% = **200%** ✅
- ROAS = 30,000 / 10,000 = **3.0** ✅

## 📈 Воронка продаж

### CTR (Click-Through Rate)

**Формула:**
```
CTR = (Клики / Показы) × 100%
```

**Что показывает:**
- Эффективность визуала и заголовка карточки
- Привлекательность товара в поисковой выдаче

**Бенчмарки:**
- **< 2%** — проблема (плохой визуал, неинтересный заголовок)
- **2-7%** — норма (средний результат)
- **7-10%** — хорошо (привлекательная карточка)
- **> 10%** — отлично (очень привлекательная карточка)

**Пример:**
- Показы: 10,000
- Клики: 500
- CTR = 500 / 10,000 × 100% = **5%** ✅ Норма

---

### Конверсия в корзину (Add to Cart Rate)

**Формула:**
```
CR корзина = (Добавления в корзину / Клики) × 100%
```

**Что показывает:**
- Качество карточки товара (фото, описание, отзывы, цена)
- Убедительность предложения

**Бенчмарки:**
- **< 5%** — слабая карточка (плохие фото, мало отзывов, высокая цена)
- **5-10%** — средняя (можно улучшить)
- **10-15%** — хорошая (качественная карточка)
- **> 15%** — отличная (сильная карточка)

**Пример:**
- Клики: 500
- Добавления в корзину: 50
- CR корзина = 50 / 500 × 100% = **10%** ✅ Хорошо

---

### Конверсия в заказ (Purchase Rate)

**Формула:**
```
CR заказ = (Заказы / Клики) × 100%
```

**Альтернативная:**
```
CR заказ = (Заказы / Добавления в корзину) × 100%
```

**Бенчмарки (от кликов):**
- **< 1%** — очень низкая
- **1-3%** — средняя
- **3-5%** — хорошая
- **> 5%** — отличная

**Бенчмарки (от корзины):**
- **< 20%** — низкая (много отваливается на этапе оформления)
- **20-40%** — средняя
- **40-60%** — хорошая
- **> 60%** — отличная

---

### Выкупаемость (Redemption Rate)

**Формула:**
```
Выкупаемость = (Выкупленные заказы / Все заказы) × 100%
```

**Что показывает:**
- Качество товара vs ожиданий покупателя
- Соответствие товара описанию и фото

**Бенчмарки:**
- **< 50%** — критично (товар не соответствует ожиданиям)
- **50-70%** — ниже нормы (нужно улучшить качество/описание)
- **70-85%** — норма
- **85-95%** — хорошо
- **> 95%** — отлично

**Влияние:**
- Низкая выкупаемость → штрафы WB → снижение позиций в поиске

**Пример:**
- Заказов: 100
- Выкупили: 80
- Выкупаемость = 80 / 100 × 100% = **80%** ✅ Норма

## ⏱️ Операционные метрики

### Оборачиваемость товара

**Формула:**
```
Оборачиваемость (дни) = Остаток на складе (шт) / Средние продажи в день (шт)
```

**Что показывает:**
- За сколько дней продастся текущий остаток
- Эффективность управления запасами

**Бенчмарки:**
- **< 30 дней** — отлично (быстро продается)
- **30-60 дней** — хорошо (нормальная скорость)
- **60-90 дней** — требует внимания (медленно продается)
- **> 90 дней** — проблемный товар (застой на складе)

**Пример:**
- Остаток: 200 шт
- Средние продажи: 5 шт/день
- Оборачиваемость = 200 / 5 = **40 дней** ✅ Хорошо

**Рекомендации по оборачиваемости:**
- **< 30 дней** → можно увеличить остаток (не будет дефицита)
- **> 90 дней** → снизить цену или запустить акцию (избавиться от застоя)

---

### Скорость продаж (Velocity)

**Формула:**
```
Velocity = Продажи за последние 7 дней / 7
```

**Что показывает:**
- Текущая скорость продаж (средние продажи в день)

**Использование:**
- Для расчета оборачиваемости
- Для планирования поставок
- Для прогноза выручки

## 💰 Точка безубыточности

**Формула:**
```
BEP (шт) = Постоянные расходы / (Цена - Переменные расходы на 1 ед)
```

**Расшифровка:**
- **Постоянные расходы** = Реклама за период, аренда офиса, зарплаты и т.д.
- **Переменные расходы** = Себестоимость + Комиссия + Логистика + Хранение (на 1 единицу)

**Что показывает:**
- Сколько нужно продать единиц чтобы выйти в ноль

**Пример:**
- Постоянные расходы: 50,000₽/месяц
- Цена продажи: 1,000₽
- Переменные расходы: 650₽/шт
- BEP = 50,000 / (1,000 - 650) = **143 шт/месяц**

## 📐 Дополнительные формулы

### Средний чек

**Формула:**
```
Средний чек = Выручка / Количество заказов
```

### Unit Economics (прибыль на 1 единицу)

**Формула:**
```
Unit Profit = Цена - (Себестоимость + Комиссия + Логистика + Хранение + Реклама на 1 ед + Налог)
```

### LTV (Lifetime Value)

**Формула (упрощённая для WB):**
```
LTV = Средний чек × Среднее кол-во покупок на клиента
```

**Примечание:** На WB сложно отслеживать повторные покупки конкретного клиента, но можно оценивать через общую статистику.
