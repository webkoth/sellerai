# WB Supply Chain — Логистика Wildberries

Skill для управления поставками и логистикой на Wildberries.

## Модели работы с WB

### FBO (Fulfillment by Operator)

**Описание:** Товар хранится на складах WB

| Плюсы | Минусы |
|-------|--------|
| Быстрая доставка покупателю | Платное хранение |
| Больше доверия покупателей | Нужен запас на складе |
| Выше в выдаче | Сложнее вернуть товар |
| Участие в акциях | Комиссия за логистику |

**Когда использовать:**
- Основная модель для большинства товаров
- Стабильный спрос
- Можно прогнозировать продажи

### FBS (Fulfillment by Seller)

**Описание:** Товар хранится у продавца, отправляется по заказу

| Плюсы | Минусы |
|-------|--------|
| Нет затрат на хранение | Дольше доставка |
| Гибкость по остаткам | Ниже в выдаче |
| Легко тестировать новые товары | Нужно самому комплектовать |
| Контроль качества | Больше работы |

**Когда использовать:**
- Тестирование нового товара
- Габаритные товары
- Товары с низким спросом
- Уникальные товары под заказ

### Сравнение моделей

| Критерий | FBO | FBS |
|----------|-----|-----|
| Скорость доставки | 1-3 дня | 3-7 дней |
| Позиции в выдаче | Выше | Ниже |
| Затраты на хранение | Есть | Нет |
| Контроль остатков | WB | Продавец |
| Комплектация | WB | Продавец |

| Хабаровск | Хабаровский край | Дальний Восток |

### Стратегия распределения

**Для старта:**
1. Основной склад (Коледино/Подольск) — 70%
2. Региональный склад (по ЦА) — 30%

**Для масштабирования:**
- Добавлять склады по мере роста продаж в регионе
- Анализировать географию заказов

### Факторы выбора склада

1. **География продаж** — где основные покупатели
2. **Срок доставки** — влияет на конверсию
3. **Стоимость логистики** — разная для разных складов
4. **Наличие приёмки** — очередь на приёмку

## Расчёт оптимальных остатков

### Формула запаса

```
Запас = Средние продажи/день × Дней до следующей поставки × Коэффициент безопасности
```

**Коэффициент безопасности:**
- Стабильный спрос: 1.2
- Умеренные колебания: 1.5
- Высокая волатильность: 2.0

### Пример расчёта

- Продажи: 10 шт/день
- Срок поставки: 14 дней
- Коэффициент: 1.3
- **Запас = 10 × 14 × 1.3 = 182 шт**

### Расчёт точки заказа

```
Точка заказа = Средние продажи/день × Срок поставки + Страховой запас
```

**Пример:**
- Продажи: 10 шт/день
- Срок поставки: 7 дней
- Страховой запас: 20 шт
- **Точка заказа = 10 × 7 + 20 = 90 шт**

Когда остаток < 90 шт — делать заказ!

## Оборачиваемость

### Формула оборачиваемости

```
Оборачиваемость (дни) = Средний остаток / Средние продажи в день
```

### Бенчмарки оборачиваемости

| Категория | Норма | Хорошо | Проблема |
|-----------|-------|--------|----------|
| Одежда | 30-60 дней | < 30 дней | > 90 дней |
| Обувь | 45-90 дней | < 45 дней | > 120 дней |
| Электроника | 30-45 дней | < 30 дней | > 60 дней |
| Товары для дома | 30-60 дней | < 30 дней | > 90 дней |

### Что делать при высокой оборачиваемости

| Оборачиваемость | Действие |
|-----------------|----------|
| > 90 дней | Снизить цену, участвовать в акциях |
| > 120 дней | Распродажа, вывод из ассортимента |
| > 180 дней | Срочный вывод (платное хранение!) |

## Планирование поставок

### Чек-лист перед поставкой

- [ ] Рассчитан оптимальный объём
- [ ] Выбран склад
- [ ] Проверена очередь на приёмку
- [ ] Товар промаркирован
- [ ] Упаковка соответствует требованиям
- [ ] Создана поставка в ЛК
- [ ] Распечатаны этикетки

### Типы поставок

| Тип | Описание | Когда использовать |
|-----|----------|-------------------|
| Короба | Товар в коробах | Мелкий товар |
| Монопаллеты | Один товар на паллете | Большие партии одного SKU |
| Микс-паллеты | Разные товары на паллете | Несколько SKU |

### Ошибки при поставках

| Ошибка | Последствие | Как избежать |
|--------|-------------|--------------|
| Неправильная маркировка | Отказ в приёмке | Проверять штрихкоды |
| Несоответствие упаковки | Штраф, отказ | Читать требования WB |
| Ошибка в количестве | Расхождения | Двойная проверка |
| Не тот склад | Потеря времени | Проверять адрес |

## Стоимость логистики

### Компоненты затрат

1. **Логистика до склада WB** — доставка поставки
2. **Логистика до покупателя** — комиссия WB
3. **Хранение** — за м³ в сутки
4. **Обратная логистика** — возвраты

### Оптимизация затрат

| Затрата | Как оптимизировать |
|---------|-------------------|
| Доставка до склада | Объединять поставки, выбирать ближний склад |
| Хранение | Не держать избыточный запас |
| Логистика покупателю | Уменьшать габариты упаковки |
| Возвраты | Улучшать описание, фото |
