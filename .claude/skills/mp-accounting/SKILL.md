# MP Accounting — Бухгалтерия маркетплейсов

Skill для бухгалтерского и налогового учёта селлеров на маркетплейсах.

## Виды налогоплательщиков

| Тип | Особенности на МП |
|-----|-------------------|
| **ООО** | Полный доступ, полный бухучёт |
| **ИП** | Полный доступ, упрощённый учёт |
| **Самозанятый (НПД)** | Только товары собственного производства! |

**Важно:** Самозанятые НЕ могут перепродавать товары на маркетплейсах.

## Системы налогообложения

### Сравнение систем

| Система | Ставка | Для кого | Сложность учёта |
|---------|--------|----------|-----------------|
| **УСН "Доходы"** | 6% | Большинство селлеров | Низкая |
| **УСН "Доходы-Расходы"** | 15% (5-15%) | При высоких расходах | Средняя |
| **ОСНО** | 20% НДС + налог на прибыль | Крупные компании | Высокая |
| **НПД** | 4-6% | Самозанятые (только своё производство) | Минимальная |
| **Патент (ПСН)** | ❌ | Не применим к МП | — |

### УСН "Доходы" 6%

**Преимущества для селлеров:**
- Простой учёт
- Не нужно подтверждать расходы
- Минимум документов
- Налог можно уменьшить на страховые взносы (до 50% для ООО/ИП с работниками, до 100% для ИП без работников)

**Формула:**
```
Налог = Доход × 6% - Уплаченные страховые взносы
```

### УСН "Доходы-Расходы" 15%

**Когда выгоднее:**
- Расходы > 60% от доходов
- Все расходы документально подтверждены
- Есть товарный учёт

**Формула:**
```
Налог = (Доходы - Расходы) × 15%
```

**Минимальный налог:** 1% от доходов (если расчётный налог меньше)

### Рекомендация по выбору

| Доля расходов | Рекомендуемая система |
|---------------|----------------------|
| < 60% | УСН "Доходы" 6% |
| 60-70% | Считать оба варианта |
| > 70% | УСН "Доходы-Расходы" 15% |

## Страховые взносы ИП

### Фиксированные взносы (2024)

| Взнос | Сумма | Срок уплаты |
|-------|-------|-------------|
| ОМС + ОПС | 49 500₽ | до 31 декабря |

**Важно:** Взносы платятся даже при отсутствии доходов!

### Дополнительные взносы 1%

```
Доп. взнос = (Доход - 300 000₽) × 1%
```

| Система | База для расчёта 1% |
|---------|---------------------|
| УСН "Доходы" | Весь доход |
| УСН "Доходы-Расходы" | Доход минус расходы |

**Максимум доп. взносов (2024):** 277 571₽

**Срок уплаты:** до 1 июля следующего года

### Пример расчёта

**ИП на УСН "Доходы", доход 2 000 000₽:**
```
Фиксированные взносы: 49 500₽
Доп. взнос: (2 000 000 - 300 000) × 1% = 17 000₽
Итого взносов: 66 500₽

Налог до вычета: 2 000 000 × 6% = 120 000₽
Налог к уплате: 120 000 - 66 500 = 53 500₽
```

## Агентская схема маркетплейсов

Все маркетплейсы работают по **агентской схеме**:
- Маркетплейс = Агент (комиссионер)
- Селлер = Принципал (комитент)
- Маркетплейс продаёт товар от своего имени, но за счёт селлера

### Документы агентской схемы

| Документ | Назначение | Частота |
|----------|------------|---------|
| **Отчёт комиссионера** | Отчёт о продажах и комиссиях | Еженедельно/ежемесячно |
| **УПД** | Подтверждение услуг | По факту |
| **Акт сверки** | Сверка взаиморасчётов | Ежемесячно/ежеквартально |

### Отчёт комиссионера

**Содержит:**
- Сумма реализации (выручка)
- Комиссия маркетплейса
- Логистика
- Хранение
- Прочие услуги
- Сумма к выплате селлеру

**В 1С:** Документ "Отчёт комиссионера о продажах"

## Настройка 1С для маркетплейсов

### Обязательные настройки

**Главная → Настройки → Функциональность:**

1. **Раздел "Расчёты":**
   - ✅ Управление зачётом авансов
   - ✅ Факторинговые операции
   - ✅ Выплаты самозанятым (если есть)

2. **Раздел "Торговля":**
   - ✅ Комиссионная торговля
   - ✅ Продажа через комиссионеров (агентов)

3. **Раздел "Расчёты":**
   - ✅ Учёт по договорам

### Договоры в 1С

Для каждого маркетплейса создаём **2 договора**:

1. **Агентский договор**
   - Вид: "С комиссионером (агентом) на продажу"
   - Для учёта продаж

2. **Договор с поставщиком (услуг)**
   - Для учёта услуг маркетплейса
   - Логистика, хранение, реклама

## ЭДО (Электронный документооборот)

### Когда нужен ЭДО

| Система | ЭДО обязателен? |
|---------|-----------------|
| ООО на ОСНО | ✅ Да |
| ООО на УСН | Желательно |
| ИП на УСН "Доходы-Расходы" | Желательно |
| ИП на УСН "Доходы" | ❌ Не обязателен |

### Операторы ЭДО для МП

- Контур.Диадок
- СберКорус
- СБИС
- Тензор

## КУДиР (Книга учёта доходов и расходов)

**Обязательна для:** ИП и ООО на УСН

### Структура для маркетплейсов

| Дата | Документ | Доход | Расход |
|------|----------|-------|--------|
| XX.XX | Отчёт комиссионера №... | Сумма реализации | — |
| XX.XX | УПД услуги МП | — | Комиссия + логистика |

**Важно:** Доход признаётся по дате отчёта комиссионера, не по дате поступления денег!

## Специфика по маркетплейсам

### Wildberries

- Отчёт комиссионера: еженедельно
- Выплаты: 2 раза в неделю
- ЭДО: через личный кабинет

### Ozon

- Отчёт комиссионера: еженедельно
- Факторинг: возможен (вывод денег чаще)
- ЭДО: Диадок, СберКорус

### Яндекс.Маркет

- Отчёт агента: ежемесячно
- Выплаты: по графику
- ЭДО: собственная система

## Частые ошибки

| Ошибка | Последствие | Как избежать |
|--------|-------------|--------------|
| Не платить взносы без дохода | Штрафы, пени | Платить фиксированные взносы |
| Патент для МП | Доначисление налогов | Использовать УСН |
| Нет отчёта комиссионера | Проблемы с учётом | Запрашивать у МП |
| Дата дохода по оплате | Неверный учёт | По дате отчёта комиссионера |
