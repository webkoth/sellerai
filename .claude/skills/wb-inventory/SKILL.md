# WB Inventory — Остатки товаров Wildberries

Skill для получения остатков товаров на складах Wildberries (FBO) с ценами и скидками.

## Требования

### API-токен

Создать токен в личном кабинете WB: [Интеграции по API](https://seller.wildberries.ru/api-integrations)

Требуемые категории токена:
- **Статистика** — для получения остатков
- **Цены и скидки** — для получения цен

### Настройка токена

**Вариант 1: Файл .env в проекте** (рекомендуется)

```bash
# Создать .env в корне проекта
echo 'WB_API_TOKEN="your_token_here"' > .env

# Добавить в .gitignore
echo ".env" >> .gitignore
```

**Вариант 2: Переменная окружения**

```bash
export WB_API_TOKEN="your_token_here"
```

**Вариант 3: Файл токена**

```bash
echo "your_token_here" > ~/.wb_token
chmod 600 ~/.wb_token
```

## Использование

### Базовый запуск

```bash
python3 scripts/get_stock.py
```

### Параметры

| Параметр | Описание |
|----------|----------|
| `--format`, `-f` | Формат вывода: `markdown` (по умолчанию) или `json` |
| `--warehouses`, `-w` | Показывать названия складов |
| `--min-quantity`, `-q` | Минимальный остаток для фильтрации (по умолчанию: 1) |
| `--output`, `-o` | Путь к файлу для сохранения результата |

### Примеры

Вывести товары с остатком >= 5:
```bash
python3 scripts/get_stock.py --min-quantity 5
```

Сохранить в JSON:
```bash
python3 scripts/get_stock.py --format json --output stock.json
```

Показать склады:
```bash
python3 scripts/get_stock.py --warehouses
```

## Формат вывода

Markdown-таблица содержит:

| Поле | Описание |
|------|----------|
| Название | Категория товара (subject) |
| Артикул | Артикул поставщика или nmId |
| Кол-во | Суммарное количество на всех складах |
| Цена | Цена до скидки |
| Цена со скидкой | Финальная цена для покупателя |
| Скидка | Размер скидки в % |

## API Endpoints

Скрипт использует следующие endpoints:

### Statistics API
- `GET /api/v1/supplier/stocks` — остатки на складах WB
- Host: `statistics-api.wildberries.ru`
- Лимит: 1 запрос/минуту

### Prices API  
- `GET /api/v2/list/goods/filter` — цены и скидки
- Host: `discounts-prices-api.wildberries.ru`
- Пагинация: limit/offset

## Ошибки

| Код | Причина | Решение |
|-----|---------|---------|
| 401 | Невалидный токен | Проверить токен, обновить если истёк |
| 403 | Нет доступа к категории | Добавить категорию при создании токена |
| 429 | Превышен лимит запросов | Подождать и повторить |
