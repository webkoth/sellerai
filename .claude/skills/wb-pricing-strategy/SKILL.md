## 🎯 Контекст применения

### Когда пересматривать ценообразование

**Критические моменты:**
1. **Запуск нового товара** — установить стартовую цену для набора позиций
2. **Изменение себестоимости** — повысилась закупочная цена или логистика
3. **Конкуренты изменили цены** — упали на 20%+ или повысили
4. **Падение продаж** — возможно цена стала неконкурентной
5. **Низкая маржа (< 20%)** — текущая цена не покрывает расходы
6. **Акции WB** — требуется участие со скидкой
7. **Сезонность** — начало/конец сезона

### Принципы ценообразования на WB

**Правило 1: Цена определяет позицию**
- WB ранжирует товары по цене внутри категории
- Снижение цены → улучшение позиций → рост продаж

**Правило 2: СПП съедает маржу**
- СПП может достигать 35% от цены
- Нужно закладывать СПП в ценообразование

**Правило 3: Маржинальность > 20% обязательна**
- При маржинальности < 20% любые колебания (возвраты, штрафы) = убытки
- Оптимально: 25-40% маржи

## Структура цены на WB

**Компоненты цены:** См. `.claude/knowledge/formulas/wb-costs.md`

**Формула минимальной цены:**
```
Минимальная цена = (Себестоимость + Фиксированные расходы) / (1 - % расходов - Целевая маржа%)
```

**Где:**
- Себестоимость = Закупка + Доставка до склада
- Фиксированные расходы = Логистика WB (₽/шт)
- % расходов = Комиссия WB% + Хранение% + Налог%
- Целевая маржа% = Желаемая прибыль (рекомендуется 20-30%)

**Пример:**
- Себестоимость: 500₽
- Логистика: 100₽
- Комиссия WB: 15%, Хранение: 2%, Налог: 6% = 23%
- Целевая маржа: 20%
- Минимальная цена = (500 + 100) / (1 - 0.23 - 0.20) = 600 / 0.57 = **1053₽**

## СПП (Скидка Постоянного Покупателя)

### Что такое СПП

СПП — скидка, которую WB даёт покупателям за счёт продавца. Применяется автоматически в зависимости от цены товара.

### Диапазоны СПП (2024)

| Цена товара | СПП |
|-------------|-----|
| до 500₽ | 0-17% |
| 500-1000₽ | 17-25% |
| 1000-2000₽ | 25-30% |
| 2000₽+ | до 35% |

**Важно:** Диапазоны регулярно меняются!

### Как использовать СПП

**Пример:**
- Цена 290₽ → СПП 17% → Цена для клиента **240₽**
- Цена 320₽ → СПП 25% → Цена для клиента **240₽**

При одинаковой цене для покупателя вы заработаете больше с ценой 320₽!

**Рекомендация:** Следить за СПП ежедневно и корректировать цены

## Стратегии ценообразования

### По этапу жизненного цикла

| Этап | Стратегия | Цена |
|------|-----------|------|
| **Запуск** | Низкая цена для набора продаж | -10-20% от рынка |
| **Рост** | Постепенное повышение | К рыночной |
| **Зрелость** | Максимальная прибыль | Рыночная или выше |
| **Спад** | Распродажа остатков | Снижение |

### По сезону

| Этап сезона | Стратегия |
|-------------|-----------|
| **До сезона** | Набор позиций, акции, низкая цена |
| **В сезон** | Максимальная цена, без акций |
| **Конец сезона** | Распродажа, активные акции |

## Работа с акциями WB

### Факторы решения об участии

1. **Стадия жизненного цикла товара**
2. **Этап сезона**
3. **Остатки и оборачиваемость**
4. **Дата следующей поставки**
5. **Жёсткость условий акции**

### Когда участвовать в акциях

| Ситуация | Участвовать? | Цель |
|----------|--------------|------|
| Запуск товара | Да | Набор продаж и позиций |
| Много остатков | Да | Увеличить оборачиваемость |
| Падение позиций | Да | Вернуться в ТОП |
| Перед поставкой | Да | Освободить место |
| Конец сезона | Да | Распродать остатки |
| Пик сезона | Нет | Максимальная прибыль |
| Мало остатков | Нет | Сохранить маржу |

### Типы акций WB

| Тип акции | Описание | Рекомендация |
|-----------|----------|--------------|
| **Тематические** | День рождения WB, 11.11 и т.д. | Участвовать (трафик) |
| **Категорийные** | На категорию товаров | По ситуации |
| **Принудительные** | WB требует участия | Оценить условия |
| **Распродажи** | Сезонные распродажи | В конце сезона |

### Влияние акций на выдачу

- **Вход в акцию** → временное падение позиций
- **Во время акции** → рост продаж, рост позиций
- **Выход из акции** → можно потерять позиции

**Лайфхак:** Выйти из акции и зайти снова — встать выше в выдаче

## Расчёт цены для акции

### Формула минимальной акционной цены

```
Мин. акц. цена = Себестоимость + Комиссия + Логистика + Мин. прибыль
```

**Пример:**
- Себестоимость: 500₽
- Комиссия WB (15%): 75₽
- Логистика: 100₽
- Минимальная прибыль (5%): 25₽
- **Минимальная цена акции: 700₽**

### Что отслеживать

| Параметр | Частота | Инструмент |
|----------|---------|------------|
| Цены конкурентов | Ежедневно | MPStats |
| СПП на товарах | Ежедневно | WB |

### Реакция на изменения

| Ситуация | Действие |
|----------|----------|
| Конкурент снизил цену | Оценить необходимость снижения |
| Изменился СПП | Скорректировать цену |
| Падение позиций | Рассмотреть участие в акции |
| Рост спроса | Повысить цену |

## Типичные ошибки

| Ошибка | Последствие | Решение |
|--------|-------------|---------|
| Завышенная цена на старте | Нет продаж, нет позиций | Начинать ниже рынка |
| Участие во всех акциях | Потеря маржи | Выбирать стратегически |
| Игнорирование СПП | Недополученная прибыль | Следить ежедневно |
| Резкое повышение цены | Падение продаж | Повышать плавно |
| Низкая цена в пик сезона | Упущенная прибыль | Максимизировать в сезон |
