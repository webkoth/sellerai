# WB Competitor Analysis — Анализ конкурентов Wildberries

Skill для анализа конкурентов и выявления конкурентных преимуществ.

## Методология анализа

### Кого анализировать

1. **Прямые конкуренты** — аналогичный товар в той же категории
2. **ТОП-10 по ключевому запросу** — лидеры выдачи
3. **Растущие конкуренты** — новички с хорошей динамикой

**Рекомендация:** Анализировать 3-5 конкурентов

### Критерии оценки конкурентов

| # | Критерий | Что анализируем | Где смотреть |
|---|----------|-----------------|--------------|
| 1 | Ссылка на товар | Сохранить для мониторинга | WB |
| 2 | Рейтинг и отзывы | Оценка, негативные отзывы, проблемы | WB карточка |
| 3 | Размерный ряд | Какие размеры продаются лучше | WB + MPStats |
| 4 | Состав/материал | Отличия от нашего товара | WB карточка |
| 5 | Фотоконтент | Ракурсы, инфографика, видео | WB карточка |
| 6 | Цена | Сравнение, динамика изменений | MPStats |
| 7 | Раскладка по складам | На какие склады отгружен | MPStats |
| 8 | Категории | В какие категории попадает | MPStats |
| 9 | Характеристики | Заполненность, отличия | MPStats |
| 10 | Запросы (ТОП) | По каким запросам в топе | MPStats |
| 11 | Коллекция | Новая/старая коллекция | WB карточка |

## Детальный анализ по критериям

### 1. Рейтинг и отзывы

**Анализируем:**
- Общий рейтинг (5.0 / 4.5+ / <4.5)
- Количество отзывов
- Негативные отзывы — ищем слабые стороны товара
- Положительные отзывы — ищем что повторить

**Вывод:** Что клиенты ценят и за что ругают

### 2. Размерный ряд (для одежды)

**Анализируем:**
- Какие размеры есть у конкурента
- Какие размеры продаются лучше всего
- Есть ли у нас эти размеры

**Вывод:** Дозаказать недостающие популярные размеры

### 3. Фотоконтент

**Анализируем:**
- Порядок фото в карточке
- Ракурсы съёмки
- Наличие модели
- Качество инфографики
- Наличие и качество видео

**Вывод:** Что взять для своей карточки

### 4. Цена

**Анализируем:**
- Текущая цена vs наша
- Динамика изменения цен
- Участие в акциях
- СПП на товаре

**Вывод:** Корректировка ценовой стратегии

### 5. Раскладка по складам

**Анализируем:**
- На каких складах есть товар
- Сроки доставки в ключевых регионах
- Наличие товара (не OOS)

**Вывод:** Куда дополнительно отгрузить

### 6. Категории и характеристики

**Анализируем:**
- В какие категории попадает товар
- Какие характеристики заполнены
- По каким фильтрам находится

**Вывод:** Доработать характеристики для попадания в нужные категории

### 7. Поисковые запросы

**Анализируем:**
- По каким запросам в ТОП-10
- Какие ключи используются
- Позиции по основным запросам

**Вывод:** Добавить эффективные ключи конкурентов

## Частые находки при анализе

| Проблема | Решение |
|----------|---------|
| Конкуренты с лучшими фото | Обновить фотоконтент |
| Конкуренты дешевле | Пересмотреть цену или добавить ценность |
| Конкуренты на большем числе складов | Расширить географию поставок |
| Конкуренты с бóльшим числом размеров | Расширить размерный ряд |
| Конкуренты в бóльшем числе категорий | Доработать характеристики |
| Конкуренты с видео | Добавить видео |
