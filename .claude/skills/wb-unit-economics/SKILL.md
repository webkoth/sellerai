## 🎯 Контекст применения

### Когда использовать этот skill

**Основные сценарии:**
1. **Оценка нового товара** — понять выгоден ли товар до запуска
2. **Анализ текущих товаров** — выявить убыточные позиции
3. **Принятие решений по акциям** — считать юнит-экономику со скидкой
4. **Планирование закупок** — учитывать оборачиваемость и маржу
5. **ABC-анализ ассортимента** — сегментировать товары по прибыльности

### Что учитывать при расчётах

**Критично важно:**
- ✅ Учитывать ВСЕ расходы (комиссии + логистика + хранение + реклама + налог)
- ✅ Делить расходы на рекламу на количество продаж (реклама на 1 ед)
- ✅ Использовать актуальные ставки комиссий WB для вашей категории
- ✅ Считать маржинальность от выручки, а не от себестоимости
- ✅ Учитывать возвраты (снижают реальную выручку на 5-15%)

**Категорийная специфика:**
- **Одежда/Обувь** — высокая доля возвратов, учитывать штрафы за низкую выкупаемость
- **Электроника** — низкая комиссия (12-15%), но высокая конкуренция по цене
- **Косметика** — высокая комиссия (18-22%), но возможна высокая маржа
- **Продукты** — короткий срок годности, критична оборачиваемость < 20 дней
- **Крупногабарит** — логистика может быть 200-500₽, учитывать в цене

## Пример расчёта юнит-экономики

### Исходные данные
- Цена продажи: 1000₽
- Себестоимость: 200₽
- Категория: Одежда (комиссия 20%)

### Расчёт

| Статья | Сумма | % от выручки |
|--------|-------|--------------|
| Выручка | 1000₽ | 100% |
| Себестоимость | -200₽ | 20% |
| Комиссия WB (20%) | -200₽ | 20% |
| Логистика | -100₽ | 10% |
| Хранение | -15₽ | 1.5% |
| Реклама | -80₽ | 8% |
| Налог УСН (7%) | -70₽ | 7% |
| **Прибыль** | **335₽** | **33.5%** |

### Метрики
- **Маржинальность:** 33.5%
- **Рентабельность:** 335 / 200 = 167.5%
- **Точка безубыточности:** если маржа < 0

## Акционная юнит-экономика

При участии в акциях WB необходимо пересчитывать юнит-экономику с учётом сниженной цены.

### Факторы принятия решения по акции

1. **Стадия цикла товара** — новинка, рост, зрелость, спад
2. **Этап сезона** — начало, пик, конец
3. **Остатки и оборачиваемость** — сколько товара и как быстро продаётся
4. **Дата подсортировки** — когда приедет новая партия
5. **Жёсткость акции** — требования WB по скидке

### Когда участвовать в акциях

| Ситуация | Рекомендация |
|----------|--------------|
| Высокая оборачиваемость (< 30 дней) | Не участвовать, продаётся и так |
| Оборачиваемость 60-90 дней | Рассмотреть участие |
| Оборачиваемость > 90 дней | Участвовать для ускорения |
| Конец сезона | Участвовать для распродажи |
| Новинка без позиций | Участвовать для набора продаж |

## ❌ Типичные ошибки при расчётах

### Ошибка 1: Не учитывают возвраты
**Проблема:** Считают выручку по заказам, а не по выкупам
**Последствия:** Завышение маржи на 5-15%
**Решение:** Использовать формулу `Выручка = Заказы × Выкупаемость × Цена`

### Ошибка 2: Забывают про хранение
**Проблема:** Не учитывают стоимость хранения на складах WB
**Последствия:** При низкой оборачиваемости хранение съедает 2-5% маржи
**Решение:** Учитывать формулу `Хранение = Объём (литры) × Тариф × Дни хранения`

**Пример:** Товар 30×20×10 см (6 литров) хранится 60 дней по 0.20₽/литр/день = 72₽
- При цене 500₽ это 14.4% от выручки 🔴
- При цене 2000₽ это 3.6% от выручки ✅

### Ошибка 3: Не делят рекламу на единицу
**Проблема:** Считают общий бюджет рекламы, но не делят на количество продаж
**Последствия:** Не видят реальную прибыль на 1 единицу
**Решение:** `Реклама на 1 ед = Общие расходы на рекламу / Количество продаж`

**Пример:** Потратили 50,000₽ на рекламу, продали 200 шт → Реклама на 1 ед = 250₽

### Ошибка 4: Игнорируют штрафы за низкую выкупаемость
**Проблема:** Не учитывают штрафы WB при выкупаемости < 50%
**Последствия:** Неожиданное снижение прибыли на 10-20%
**Решение:** Мониторить выкупаемость, улучшать соответствие товара описанию

### Ошибка 5: Считают маржу от закупки
**Проблема:** Используют формулу `(Цена - Себестоимость) / Себестоимость` вместо правильной
**Последствия:** Завышение маржинальности в 2-3 раза
**Решение:** Правильная формула `Маржинальность = (Выручка - Все расходы) / Выручка`

**Пример:**
- ❌ Неправильно: (1000 - 200) / 200 = 400% (рентабельность)
- ✅ Правильно: (1000 - 665) / 1000 = 33.5% (маржинальность)

## 📊 Интерпретация результатов

### Маржинальность

| Диапазон | Интерпретация | Действия |
|----------|---------------|----------|
| **< 15%** | 🔴 КРИТИЧНО — товар убыточный или близок к этому | Пересмотреть цену, снизить себестоимость или уйти с товара |
| **15-25%** | 🟡 НОРМАЛЬНО — работать можно, но есть риски | Оптимизировать расходы (логистика, реклама) |
| **25-40%** | 🟢 ХОРОШО — здоровая маржа | Стабильно работать, можно масштабировать |
| **> 40%** | 🟢 ОТЛИЧНО — высокомаржинальный товар | Масштабировать, увеличить закупки |

### Рентабельность вложений

| Диапазон | Интерпретация | Действия |
|----------|---------------|----------|
| **< 30%** | 🔴 НИЗКАЯ — медленный возврат инвестиций | Проверить ценообразование, снизить расходы |
| **30-50%** | 🟡 СРЕДНЯЯ — приемлемо для большинства категорий | Продолжать работу |
| **50-80%** | 🟢 ХОРОШАЯ — быстрый возврат вложений | Хороший товар для масштабирования |
| **> 80%** | 🟢 ОТЛИЧНАЯ — очень быстрый возврат | Приоритетный товар для закупок |

### Оборачиваемость

| Диапазон | Интерпретация | Действия |
|----------|---------------|----------|
| **< 30 дней** | 🟢 ОТЛИЧНО — товар продаётся очень быстро | Можно увеличить остаток, проверить нет ли дефицита |
| **30-60 дней** | 🟢 ХОРОШО — нормальная скорость продаж | Стабильно работать |
| **60-90 дней** | 🟡 ТРЕБУЕТ ВНИМАНИЯ — медленно продаётся | Запустить акцию, увеличить рекламу, снизить остаток |
| **> 90 дней** | 🔴 ПРОБЛЕМНЫЙ ТОВАР — застой на складе | Срочная распродажа со скидкой 30-50%, не докупать |

**Связь с хранением:**
- Оборачиваемость > 60 дней → хранение начинает съедать маржу
- Оборачиваемость > 90 дней → штрафные ставки хранения (0.50-1.00₽/литр/день)

```markdown
| Комиссия WB | X₽ (X%) |
| Логистика | X₽ |
| Хранение | X₽ |
| Реклама на 1 ед | X₽ |
| Налог | X₽ |
| **Прибыль на 1 ед** | **X₽** |

### Метрики
- Маржинальность: X%
- Рентабельность: X%

```
