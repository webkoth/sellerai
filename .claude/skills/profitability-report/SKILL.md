# Отчёт о прибыльности товаров Wildberries

Формирует детальный финансовый отчёт по каждому товару с расчётом прибыли, маржи и ROI.

---

## Необходимые данные

### 1. Данные о реализации (из WB API)
Получить через `wb_get_realization_report(dateFrom, dateTo)`:
- Выручка к перечислению
- Комиссии, логистика
- Продажи/возвраты

### 2. Себестоимость товаров (от пользователя)
CSV-файл с колонками: `Артикул продавца`, `СЕБЕСТОИМОСТЬ`

**Без себестоимости отчёт невозможен!**

---

## Алгоритм

### Шаг 1: Проверка данных

```
1. Ищи файл себестоимости в reports/:
   - *sebestoimost*.csv
   - *cost*.csv
   - *себестоимость*.csv

2. Если не найден — СТОП, выведи:
```

```markdown
## ⚠️ Не хватает данных для отчёта

### Отсутствует файл себестоимости

Для формирования отчёта необходим CSV-файл с себестоимостью товаров.

**Требуемый формат файла:**

| Артикул продавца | Название товара | СЕБЕСТОИМОСТЬ |
|------------------|-----------------|---------------|
| JW-NB-001 | Браслет с метеоритом | 3200 |

**Варианты:**
1. Загрузите CSV-файл в папку `reports/`
2. Или скажите "создай шаблон себестоимости" — я сгенерирую файл на основе ваших товаров
```

### Шаг 2: Загрузка и расчёт

```
1. Прочитай CSV с себестоимостью
2. Получи отчёт: wb_get_realization_report(dateFrom, dateTo)
3. Объедини по артикулу продавца
4. Рассчитай для каждого товара:
   - Розничная цена = К перечислению / 0.82
   - Комиссия WB = Розничная - К перечислению
   - Себестоимость партии = Продано × Себестоимость ед.
   - Валовая прибыль = К перечислению - Себестоимость партии
   - Маржа = Прибыль / К перечислению × 100%
   - ROI = Прибыль / Себестоимость × 100%
```

### Шаг 3: Генерация Excel

Использовать скрипт: `scripts/generate_profitability_report.py`

```bash
# Проверка файлов
.venv/bin/python3 .claude/skills/profitability-report/scripts/generate_profitability_report.py --check

# Генерация шаблона себестоимости
.venv/bin/python3 .claude/skills/profitability-report/scripts/generate_profitability_report.py \
  --generate-template \
  --data-file reports/product_profitability.json

# Генерация отчёта
.venv/bin/python3 .claude/skills/profitability-report/scripts/generate_profitability_report.py \
  --data-file reports/product_profitability.json \
  --output reports/product_profitability_report.xlsx
```

## Формат Excel-отчёта

### Лист 1: Сводка
| Показатель | Значение |
|------------|----------|
| Товаров в отчёте | X |
| Продано единиц | X |
| Выручка к перечислению | X ₽ |
| Общая себестоимость | X ₽ |
| **Валовая прибыль** | **X ₽** |
| **Средняя маржа** | **X%** |

### Лист 2: Все товары
Таблица с колонками: Артикул, Название, Продано, Розн. цена, Комиссия WB, К перечислению, Себестоимость, Прибыль, Маржа, ROI

### Лист 3: ТОП-20
20 самых прибыльных товаров

### Лист 4: Убыточные
Товары с отрицательной маржой + рекомендации

---

## Рекомендации в отчёте

```markdown
### 💡 Рекомендации

**Убыточные товары (X шт):**
- [Артикул]: Убыток X₽, маржа -X% → Поднять цену или вывести

**Низкомаржинальные (маржа < 20%):**
- [Артикул]: Маржа X% → Искать поставщика дешевле

**Топовые товары:**
- [Артикул]: Прибыль X₽, ROI X% → Увеличить остатки
```

---

## Интерпретация маржи

| Диапазон | Интерпретация |
|----------|---------------|
| **< 15%** | 🔴 Убыточный — пересмотреть цену |
| **15-25%** | 🟡 Нормально — оптимизировать расходы |
| **25-40%** | 🟢 Хорошо — стабильно работать |
| **> 40%** | 🟢 Отлично — масштабировать |
