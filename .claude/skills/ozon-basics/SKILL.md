# Ozon Basics — Основы работы на Ozon

Skill для работы с маркетплейсом Ozon.

## Кто может продавать на Ozon

| Тип продавца | Доступ | Ограничения |
|--------------|--------|-------------|
| ООО | ✅ Полный | — |
| ИП | ✅ Полный | — |
| Самозанятый | ✅ Ограниченный | Только товары собственного производства |

## Модели работы с Ozon

### FBO (Fulfillment by Ozon)

**Описание:** Товар хранится на складах Ozon

| Плюсы | Минусы |
|-------|--------|
| Быстрая доставка (1-2 дня) | Платное хранение |
| Выше в выдаче | Нужен запас на складе |
| Участие в акциях | Комиссия за логистику |
| Меньше работы | Сложнее вернуть товар |

**Когда использовать:**
- Стабильный спрос
- Товары с хорошей оборачиваемостью
- Основная модель для масштабирования

### FBS (Fulfillment by Seller)

**Описание:** Товар у продавца, доставка через Ozon

| Плюсы | Минусы |
|-------|--------|
| Нет затрат на хранение | Дольше доставка |
| Контроль остатков | Нужно комплектовать |
| Гибкость | Ниже в выдаче |

**Когда использовать:**
- Тестирование товаров
- Крупногабаритные товары
- Низкий/нестабильный спрос

### realFBS

**Описание:** Товар у продавца, доставка самостоятельно или через партнёров

| Плюсы | Минусы |
|-------|--------|
| Полный контроль доставки | Больше работы |
| Свои курьеры | Сложнее масштабировать |
| Гибкие сроки | — |

**Когда использовать:**
- Локальные продажи
- Особые условия доставки
- Хрупкие/специфические товары

### Сравнение моделей

| Критерий | FBO | FBS | realFBS |
|----------|-----|-----|---------|
| Скорость доставки | 1-2 дня | 2-5 дней | Зависит от вас |
| Хранение | На складе Ozon | У вас | У вас |
| Комплектация | Ozon | Вы | Вы |
| Доставка | Ozon | Ozon | Вы/партнёры |
| Позиции в выдаче | Выше | Средние | Ниже |

## Комиссии Ozon

### Структура комиссий

1. **Комиссия за продажу** — % от цены товара (зависит от категории)
2. **Логистика** — доставка до покупателя
3. **Последняя миля** — доставка до ПВЗ
4. **Обработка возврата** — при возврате товара
5. **Хранение** — за FBO товары

### Комиссии по категориям (примеры)

| Категория | Комиссия |
|-----------|----------|
| Электроника | 5-10% |
| Одежда | 8-15% |
| Товары для дома | 10-15% |
| Косметика | 10-15% |
| Продукты | 5-8% |

**Важно:** Актуальные комиссии смотреть в личном кабинете Ozon Seller

### Логистика FBO

| Параметр | Стоимость |
|----------|-----------|
| До 5 литров | от 50₽ |
| 5-10 литров | от 80₽ |
| 10-25 литров | от 120₽ |
| Крупногабарит | Индивидуально |

## Карточка товара

### Обязательные элементы

1. **Название** — до 200 символов
2. **Описание** — до 6000 символов
3. **Фото** — минимум 1, рекомендуется 5-10
4. **Характеристики** — все обязательные для категории
5. **Цена** — с учётом всех комиссий

### Рекомендации по контенту

**Название:**
```
[Бренд] [Тип товара] [Ключевые характеристики] [Для кого/чего]
```

**Фото:**
- Главное фото — товар на белом фоне
- Дополнительные — ракурсы, детали, в использовании
- Инфографика — преимущества, размеры
- Видео — демонстрация

**Описание:**
- Преимущества в начале
- Ключевые характеристики
- Применение
- Комплектация

### Rich-контент

Ozon поддерживает расширенное описание:
- Модули с фото
- Таблицы характеристик
- Видео
- 360° фото

## Поставки на Ozon

### Типы поставок FBO

| Тип | Описание | Когда использовать |
|-----|----------|-------------------|
| Короба | Товары в коробках | Мелкий товар |
| Монопаллеты | Один артикул на паллете | Большие партии |
| Микспаллеты | Разные артикулы | Несколько SKU |

### Алгоритм поставки

1. Создать поставку в ЛК
2. Подготовить товар (маркировка, упаковка)
3. Сформировать документы
4. Доставить на склад
5. Дождаться приёмки

### Требования к маркировке

- Штрихкод на каждой единице
- Этикетка поставки на коробе
- Штрихкод = артикул Ozon (SKU)

## Частые ошибки на Ozon

| Ошибка | Последствие | Решение |
|--------|-------------|---------|
| Слабый контент | Низкая конверсия | Качественные фото, описание |
| Неверная категория | Высокая комиссия | Проверить категорию |
| Нет остатков FBO | Потеря позиций | Следить за остатками |
| Игнорирование отзывов | Падение рейтинга | Отвечать на все отзывы |
| Завышенная цена | Нет продаж | Анализ конкурентов |
