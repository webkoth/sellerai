## 🎯 Контекст применения

### Когда запускать рекламу

**✅ Рекламу НУЖНО запускать:**
1. **Новый товар без позиций** — набрать продажи и отзывы для органического роста
2. **Высокомаржинальный товар (> 30%)** — можно позволить 10-15% на рекламу
3. **Сезонный товар в начале сезона** — захватить позиции до пика спроса
4. **Товар с CTR > 5%** — хорошая карточка, реклама окупится
5. **Конкурентная ниша** — без рекламы не выйти в топ

**❌ Рекламу НЕ нужно запускать:**
1. **Низкомаржинальный товар (< 20%)** — реклама съест всю прибыль
2. **Плохая карточка (CTR < 2%)** — сначала улучшить визуал/описание
3. **Товар уже в топ-10 органически** — продаётся и так
4. **Нет остатков** — деньги на ветер
5. **Конец сезона** — спрос падает, реклама не окупится

### Оптимальная доля рекламы от выручки

| Категория | Норма | Максимум | Критичный уровень |
|-----------|-------|----------|-------------------|
| **Одежда/Обувь** | 5-10% | 15% | > 20% убыточно |
| **Электроника** | 3-7% | 10% | > 12% убыточно |
| **Товары для дома** | 5-8% | 12% | > 15% убыточно |
| **Косметика** | 7-12% | 18% | > 20% убыточно |
| **Новинка (первый месяц)** | 10-20% | 30% | Ограничено временем |

**Правило:** Если доля рекламы > Максимума → либо снизить ставки, либо улучшить карточку, либо уйти с товара

### Автоматические кампании (рекомендуется для старта)

**Преимущества:**
- Самая низкая минимальная ставка (от 125₽ CPM)
- Автоматический подбор товаров и мест показа
- Работает во всех плейсментах сразу
- Оптимизация на основе алгоритмов WB

**Когда использовать:**
- Старт продвижения новых товаров
- Тестирование спроса
- Малый бюджет

### Реклама в поиске

**Особенности:**
- Товар на первой странице по ВСЕМ запросам, где он индексирован
- Не нужно подбирать ключевые слова вручную
- Можно исключить нерелевантные запросы
- Только на Desktop

**Когда использовать:**
- Вывод товара в топ по ключевым запросам
- Конкуренция за высокочастотные запросы
- Быстрый набор позиций

### Реклама в каталоге

**Особенности:**
- Поднимает товар на верхние позиции в категории
- Работает по системе аукциона
- Только на Desktop

**Когда использовать:**
- Конкуренция в категории
- Товары без чёткого поискового спроса
- Дополнение к поисковой рекламе

### Реклама в карточке товара

**Особенности:**
- Показ в блоке "Смотрите также" у конкурентов
- Перехват трафика конкурентов
- Работает на всех платформах

**Когда использовать:**
- Отстройка от конкурентов
- Товары-аналоги
- Ценовое преимущество

## Система аукциона

### Как работает

1. **Ставка CPM** — стоимость за 1000 показов
2. **Аукцион в реальном времени** — торги за рекламное место
3. **Выигрывает высшая ставка** — с учётом релевантности

### Формула эффективности

```
Эффективность = CTR × CR × (Выручка - Расходы на рекламу)
```

### Стратегии ставок

| Стратегия | Ставка | Когда использовать |
|-----------|--------|-------------------|
| **Агрессивная** | +30-50% от рекомендованной | Запуск, набор позиций |
| **Умеренная** | Рекомендованная | Стабильные продажи |
| **Экономная** | -20-30% от рекомендованной | Оптимизация расходов |

## 📊 Ключевые метрики рекламы

**Все формулы:** См. `.claude/knowledge/formulas/wb-metrics.md`

### Основные метрики для мониторинга

| Метрика | Что показывает | Целевое значение | Действия при отклонении |
|---------|----------------|------------------|-------------------------|
| **CTR** | Привлекательность карточки в показах | 3-7% | < 2% → улучшить главное фото |
| **CR (корзина)** | Качество карточки при клике | 5-15% | < 5% → улучшить описание, фото |
| **CR (заказ)** | Конверсия из корзины в заказ | 1-4% | < 1% → проверить цену |
| **ROI** | Окупаемость рекламы | > 100% | < 50% → снизить ставки |
| **ROAS** | Возврат с рубля рекламы | > 3.0 | < 2.0 → оптимизировать |
| **CPO** | Стоимость привлечения заказа | < 10% маржи | > 15% → проблема |

**Критичные пороги:**
- CTR < 1% → реклама впустую, отключить
- ROI < 0% 3 дня подряд → немедленно отключить кампанию
- CPO > маржинальность → каждый заказ убыточен

### Расчёт минимального бюджета

**Формула:** `Бюджет = (Целевые показы / 1000) × Ставка CPM`

**Пример планирования:**
- Хочу 10,000 показов
- Рекомендованная ставка: 200₽ CPM
- Минимальный бюджет = (10,000 / 1000) × 200 = **2,000₽**

**Правило:** Начинать с бюджета на 3-7 дней работы, затем анализировать ROI

## Стратегии запуска рекламы

### Для новых товаров

1. **День 1-3:** Автореклама со средней ставкой
2. **День 4-7:** Анализ CTR, корректировка ставок
3. **День 8-14:** Добавление поисковой рекламы по эффективным запросам
4. **День 15+:** Оптимизация по ROI

### Для товаров в ТОП-100

1. Поисковая реклама для удержания позиций
2. Реклама в карточках конкурентов
3. Минимальные ставки для поддержки

### Для сезонных товаров

1. **До сезона:** Агрессивная реклама, набор позиций
2. **В сезон:** Умеренная реклама, высокие продажи
3. **После сезона:** Минимальная реклама или отключение

## Оптимизация рекламных кампаний

### Чек-лист ежедневной оптимизации

- [ ] Проверить CTR (если < 2% — проблема с карточкой)
- [ ] Проверить расход vs заказы
- [ ] Скорректировать ставки по эффективности
- [ ] Отключить неэффективные товары/запросы
- [ ] Проверить баланс рекламного кабинета

### Когда повышать ставку

- CTR > 5%, но мало показов
- Конкуренты выше в выдаче
- Начало сезона

### Когда снижать ставку

- CTR < 2%
- ROI < 100%
- Конец сезона
- Товар уже в ТОП-10

### Когда отключать рекламу

- ROI отрицательный 3+ дней
- CTR < 1% после оптимизации карточки
- Товар закончился на складе
- Конец сезона + высокая оборачиваемость

## ❌ Типичные ошибки в рекламе

### Ошибка 1: Запуск рекламы с плохой карточкой
**Проблема:** Включают рекламу до оптимизации визуала и описания
**Последствия:** CTR < 2%, деньги тратятся впустую
**Решение:**
1. Сначала оптимизировать карточку (фото, заголовок, описание)
2. Набрать органический CTR > 3%
3. Только потом запускать рекламу

**Пример:** Товар с плохим фото → CTR 1% → Расход 10,000₽, продаж 0 🔴

### Ошибка 2: Не следят за остатками
**Проблема:** Реклама работает, но товара нет на складе
**Последствия:** Деньги на показы, заказов нет
**Решение:** Автоматизировать мониторинг остатков, отключать рекламу при остатке < 10 шт

**Пример:** Остаток 0 шт → Реклама работает → Расход 5,000₽, продаж 0 🔴

### Ошибка 3: Слишком широкий таргетинг
**Проблема:** Реклама показывается по нерелевантным запросам
**Последствия:** Низкий CTR, высокий расход, мало заказов
**Решение:**
1. Проверять поисковые запросы в отчётах
2. Добавлять в минус-слова нерелевантные
3. Фокусироваться на узкой нише

**Пример:** Продаём "платье летнее" → Показывается по запросу "платье зимнее" → CTR 0.5% 🔴

### Ошибка 4: Не анализируют ROI регулярно
**Проблема:** Запустили рекламу и забыли
**Последствия:** Слив бюджета на неэффективные кампании
**Решение:** Ежедневный мониторинг ROI, отключение кампаний с ROI < 0% за 3 дня

**Пример:** Реклама работает месяц, ROI -50% → Потеряно 50,000₽ 🔴

### Ошибка 5: Слишком высокие ставки на старте
**Проблема:** Ставят +100% от рекомендованной, чтобы "точно было показов"
**Последствия:** Бюджет сливается за 1-2 дня, ROI низкий
**Решение:** Начинать с рекомендованной ставки, повышать постепенно на 10-20%

**Пример:** Рекомендованная 150₽ → Поставили 300₽ → Бюджет 10,000₽ ушёл за 1 день → ROI 50% 🔴

### Ошибка 6: Реклама низкомаржинального товара
**Проблема:** Запускают рекламу на товар с маржой < 20%
**Последствия:** Даже при хорошем ROI реклама съедает всю прибыль
**Решение:** Считать юнит-экономику с учётом рекламы ПЕРЕД запуском

**Пример:**
- Маржа 15% (150₽ с товара за 1000₽)
- Реклама 10% (100₽)
- Итоговая прибыль: 50₽ (5%) → Не выгодно 🔴
