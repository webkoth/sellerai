# WB SEO Expert — SEO-оптимизация Wildberries

Skill для SEO-оптимизации карточек товаров на Wildberries: индексация, ранжирование, семантическое ядро.

## Ключевые термины

### Индексация
Наличие артикула в выдаче по поисковым запросам и категориям.

**Важно:** Нет индексации = Нет ранжирования

### Ранжирование
Занимаемое место в выдаче по категории или поисковому запросу среди конкурентов.

**Важно:** Нет ранжирования ≠ Нет индексации (товар может быть проиндексирован, но на низких позициях)

## Факторы ранжирования WB

### Группа 1: Релевантность

| Фактор | Описание | Вес |
|--------|----------|-----|
| **Соответствие запросу** | Насколько товар подходит под поисковый запрос | Высокий |
| **Кластер запроса (предмет)** | Правильная категория товара | Высокий |
| **Просмотр → Клик → Заказ** | Конверсионная цепочка | Высокий |
| **Близость к покупателю** | Срок доставки | Средний |

### Группа 2: Деньги

| Фактор | Описание | Вес |
|--------|----------|-----|
| **Заказы** | Количество и динамика | Критический |
| **Продажи** | Выручка и % выкупа | Критический |
| **Цена** | Конкурентность цены | Высокий |
| **Акции** | Участие в промо WB | Средний |

### Группа 3: Сервис

| Фактор | Описание | Вес |
|--------|----------|-----|
| **Оформление карточки** | Контент, видео, фото | Средний |
| **Отзывы** | Количество и рейтинг | Средний |
| **Ответы на отзывы и вопросы** | Активность продавца | Низкий |
| **Поведенческая активность** | Время на карточке, скролл | Низкий |

## Простые правила для роста позиций

1. **Динамика заказов и выручка** — главный фактор
2. **Совпадение полей карточки** с релевантными категориями и запросами
3. **Конверсия карточки** — продажи с учётом % возврата
4. **Участие в акциях** — повышает позиции
5. **Правильный заголовок** — самый конверсионный запрос + особенность товара
6. **Отсутствие ошибок** — нет повторов, тегов, перечислений, чужих категорий
7. **Характеристики** — дополнительные свойства, которые конвертируются в продажи

## Структура SEO-оптимизированной карточки

### Название товара

**Формула:**
```
[Категория/Предмет] + [Ключевой запрос] + [Особенность/УТП] + [Для кого]
```

**Пример:**
```
Топ женский укороченный с сердечком летний хлопок для девушек
```

**Правила:**
- До 60 символов (оптимально)
- Самый конверсионный запрос — первым
- Без повторов и спама
- Без тегов типа #хит #топ
- Без перечисления цветов/размеров

### Описание

**Структура:**
1. Краткое описание товара (1-2 предложения)
2. Преимущества и особенности
3. Материал и состав
4. Рекомендации по уходу
5. Ключевые запросы (естественно вписанные)

**Объём:** 500-1000 символов

### Характеристики

**Обязательные:**
- Состав
- Цвет
- Размер
- Страна производства
- Бренд

**Дополнительные (по категории):**
- Сезон
- Стиль
- Тип посадки
- Длина
- Особенности

## Работа с семантическим ядром

### Источники ключевых слов

1. **Поисковые подсказки WB** — при вводе запроса
2. **Wordstat Yandex** — частотность запросов
3. **MPStats / Wildbox** — аналитика конкурентов
4. **Отзывы покупателей** — как описывают товар
5. **Карточки конкурентов** — какие слова используют

### Типы запросов

| Тип | Пример | Конверсия |
|-----|--------|-----------|
| **Высокочастотные (ВЧ)** | "платье" | Низкая |
| **Среднечастотные (СЧ)** | "платье летнее" | Средняя |
| **Низкочастотные (НЧ)** | "платье летнее хлопок макси" | Высокая |

### Стратегия подбора

1. Собрать 50-100 релевантных запросов
2. Отсортировать по частотности
3. Выбрать 1-2 ВЧ для названия
4. Добавить 5-10 СЧ в описание
5. Использовать НЧ в характеристиках

## Ранжирование NEW карточек

Новые карточки получают временный буст от WB для "тестирования".

### Как использовать:

1. **Первые 7-14 дней** — максимальная активность
2. Запустить рекламу сразу
3. Участвовать в акциях
4. Получить первые отзывы
5. Обеспечить наличие на складах

### Критерии успеха теста:

- CTR > 5%
- Добавление в корзину > 10%
- Первые продажи в первые 3 дня

## Чек-лист SEO-аудита карточки

### Название
- [ ] Содержит основной ключевой запрос
- [ ] Длина до 60 символов
- [ ] Нет повторов и спама
- [ ] Указана особенность товара
- [ ] Соответствует категории

### Описание
- [ ] Содержит ключевые запросы
- [ ] Естественный текст (не спам)
- [ ] 500-1000 символов
- [ ] Описаны преимущества
- [ ] Указан состав/материал

### Характеристики
- [ ] Все обязательные поля заполнены
- [ ] Указаны дополнительные свойства
- [ ] Нет ошибок в значениях
- [ ] Соответствуют реальному товару

### Категория
- [ ] Правильный предмет (кластер)
- [ ] Проверена индексация в категории
- [ ] Товар виден в каталоге

### Общее
- [ ] Карточка проиндексирована по основным запросам
- [ ] Позиция отслеживается
- [ ] Нет дублей карточки

## Типичные ошибки SEO

| Ошибка | Последствие | Как исправить |
|--------|-------------|---------------|
| Спам ключами | Пессимизация | Переписать естественно |
| Неправильная категория | Нет индексации | Выбрать правильный предмет |
| Повторы в названии | Снижение позиций | Убрать дубли |
| Чужие категории в тексте | Рассеивание релевантности | Удалить нерелевантное |
| Слишком короткое описание | Меньше ключей | Расширить текст |
| Пустые характеристики | Потеря фильтров | Заполнить все поля |
