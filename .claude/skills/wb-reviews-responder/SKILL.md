# WB Reviews Responder — Управление отзывами Wildberries

Skill для работы с отзывами и вопросами покупателей на Wildberries.

## Влияние отзывов на продажи

Отзывы влияют на все этапы воронки:
- **Трафик** — рейтинг виден в выдаче
- **CTR** — количество отзывов влияет на клик
- **Корзина** — содержание отзывов влияет на решение
- **Заказ** — доверие к товару

## Критерии рейтинга

| Рейтинг | Статус | Действие |
|---------|--------|----------|
| **5.0** | Отлично | Поддерживать |
| **4.5+** | Хорошо | Работать над ростом |
| **< 4.5** | Проблема | Срочно исправлять |

**При рейтинге < 4.5 проверить:**
- Качество товара
- Упаковку
- Соответствие описанию

## Стратегия получения отзывов

### Первые отзывы (старт продаж)

**Способы получения:**
1. Самовыкупы с отзывами
2. Кэшбэк/раздачи в соцсетях
3. Просьбы друзьям и знакомым

**Требования к первым отзывам:**
- С фотографиями товара
- Раскрывают преимущества
- Создают доверие у покупателей

### Органические отзывы

**Как получать больше положительных:**

1. **Превосходить ожидания покупателя**
   - Качество выше ожиданий → выше % выкупа
   - Приятное удивление → мотивация оставить отзыв

2. **Оставлять контакты службы поддержки**
   - Вкладыш: "Свяжитесь с нами, если качество вас не устраивает"
   - Недовольные пишут в поддержку, а не в отзывы

## Работа с негативными отзывами

### Схема отработки

| Тип негатива | Действие |
|--------------|----------|
| Не по теме товара (ПВЗ, доставка) | Ответить, что не касается товара |
| Обоснованная претензия | Извиниться, предложить решение |
| Заказной негатив | Жалоба в WB |
| Брак товара | Предложить связаться для замены |

### Алгоритм ответа на негатив

1. **Ответить на негативный отзыв**
   - Поблагодарить за обратную связь
   - Выразить сожаление
   - Предложить решение

2. **Попросить удалить отзыв** (через поддержку)
   - Если проблема решена
   - Если отзыв не по теме

3. **Жаловаться на неадекватные отзывы**
   - Оскорбления
   - Нецензурная лексика
   - Заказные отзывы
   - Отзывы не о товаре

## Управление позицией отзывов

### Как поднять положительные отзывы

1. **Ставить лайки** нужным отзывам
   - Положительным
   - С фотографиями
   - Подробным

2. **Отвечать на положительные** (поднимает в топ)
   - "Спасибо за отзыв!"
   - Персонализированный ответ

### Как опустить негативные

1. **Жалоба** — опускает в списке
2. **Не лайкать** негативные отзывы
3. **Засыпать положительными** — негатив уйдёт вниз

## Работа с вопросами

### Правила ответов на вопросы

| Правило | Описание |
|---------|----------|
| **Отвечать быстро** | Проверять раз в час (уведомлений нет!) |
| **Отработать в карточке** | Один вопрос = 100 людей с тем же вопросом |
| **Рекламировать товары** | Можно ссылаться на другие свои артикулы |

### После ответа на вопрос

Если вопрос типичный — обновить карточку:
- Скорректировать описание
- Добавить фото
- Добавить информацию в инфографику

### Вопросы с негативом

**Вариант 1:** Оставить без ответа (не привлекать внимание)

**Вариант 2:** Тактика изменения ответа
1. Ответить, что разберётесь
2. Дождаться, пока покупатель увидит ответ
3. Изменить ответ на нейтральный

## Автоматизация

### Автоответы на отзывы

Можно настроить шаблоны:
- "Спасибо за отзыв, приходите снова!"
- "Рады, что товар понравился!"

**Важно:** Негативные отзывы — только вручную!

### Частота проверки

| Элемент | Частота |
|---------|---------|
| Отзывы | Раз в день |
| Вопросы | Раз в час |
| Рейтинг | Ежедневно |

## Шаблоны ответов

### На положительный отзыв

```
Благодарим за отзыв! Рады, что товар вам понравился.
Будем рады видеть вас снова в нашем магазине!
```

### На негативный отзыв (качество)

```
Благодарим за обратную связь. Приносим извинения за
доставленные неудобства. Мы учтём ваши замечания и
улучшим качество. Свяжитесь с нами для решения вопроса:
[контакт]
```

### На негативный отзыв (не по теме)

```
Спасибо за отзыв. К сожалению, вопросы работы ПВЗ/доставки
не относятся к нашему товару — это зона ответственности
Wildberries. Рекомендуем обратиться в поддержку WB.
```

### На вопрос

```
Здравствуйте! [Ответ на вопрос]. Если остались вопросы —
пишите, с удовольствием поможем!
```
