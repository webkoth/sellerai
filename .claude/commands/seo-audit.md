# SEO-аудит карточки

Используй skill `wb-seo-expert` для проведения аудита.

## Алгоритм

1. **Получение данных карточки:**
   - Запроси у пользователя или получи через API:
     - Название товара (title)
     - Описание
     - Характеристики
     - Категория (предмет)
     - Текущие позиции по ключевым запросам (если известны)

2. **Проведи аудит по чек-листу из skill:**

   **Название:**
   - [ ] Содержит основной ключевой запрос
   - [ ] Длина до 60 символов
   - [ ] Нет повторов и спама
   - [ ] Указана особенность товара
   - [ ] Соответствует категории

   **Описание:**
   - [ ] Содержит ключевые запросы
   - [ ] Естественный текст (не спам)
   - [ ] 500-1000 символов
   - [ ] Описаны преимущества
   - [ ] Указан состав/материал

   **Характеристики:**
   - [ ] Все обязательные поля заполнены
   - [ ] Указаны дополнительные свойства
   - [ ] Нет ошибок в значениях

   **Категория:**
   - [ ] Правильный предмет (кластер)
   - [ ] Товар виден в каталоге

3. **Выведи результат в формате:**

```markdown
## SEO-аудит: [Название товара]

### Общая оценка: X/10 [🟢/🟡/🔴]

### Анализ элементов

| Элемент | Оценка | Статус | Проблема |
|---------|--------|--------|----------|
| Название | X/10 | 🟢/🟡/🔴 | [если есть] |
| Описание | X/10 | 🟢/🟡/🔴 | [если есть] |
| Характеристики | X/10 | 🟢/🟡/🔴 | [если есть] |
| Категория | X/10 | 🟢/🟡/🔴 | [если есть] |

### Найденные проблемы

1. **[Проблема]** — [Влияние на ранжирование]
   - Как исправить: [конкретное действие]

### Рекомендации по улучшению

#### Название
**Текущее:** [текущее название]
**Рекомендуемое:** [оптимизированное название]

#### Ключевые слова для добавления
- [ключ 1] — частотность ~X
- [ключ 2] — частотность ~X
- [ключ 3] — частотность ~X

#### Характеристики для заполнения
- [характеристика 1]
- [характеристика 2]

### Приоритет действий

| # | Действие | Влияние | Срочность |
|---|----------|---------|-----------|
| 1 | [Действие] | Высокое | Срочно |
| 2 | [Действие] | Среднее | Важно |
| 3 | [Действие] | Низкое | Желательно |
```

## Дополнительные проверки

- Проверь на типичные ошибки из skill (спам, повторы, чужие категории)
- Предложи формулу названия: `[Категория] + [Ключевой запрос] + [Особенность] + [Для кого]`
- Если возможно, сравни с топовыми конкурентами по запросу
