# Анализ конкурентов

Используй skill `wb-competitor-analysis` для проведения анализа.

## Алгоритм

1. **Определение объекта анализа:**
   - Если указан артикул — найди ТОП-5 конкурентов по категории
   - Если указан ключевой запрос — найди ТОП-10 в выдаче
   - Если ничего не указано — запроси у пользователя

2. **Сбор данных о конкурентах (запроси ссылки у пользователя):**

   | # | Критерий | Где смотреть |
   |---|----------|--------------|
   | 1 | Ссылка на товар | WB |
   | 2 | Рейтинг и отзывы | WB карточка |
   | 3 | Размерный ряд | WB + MPStats |
   | 4 | Состав/материал | WB карточка |
   | 5 | Фотоконтент | WB карточка |
   | 6 | Цена и динамика | MPStats |
   | 7 | Раскладка по складам | MPStats |
   | 8 | Категории | MPStats |
   | 9 | Характеристики | MPStats |
   | 10 | ТОП-запросы | MPStats |
   | 11 | Коллекция | WB карточка |

3. **Анализ по каждому критерию:**

   **Рейтинг и отзывы:**
   - Общий рейтинг (5.0 / 4.5+ / <4.5)
   - Количество отзывов
   - Основные темы негатива — слабые стороны конкурента
   - Основные темы позитива — что повторить

   **Фотоконтент:**
   - Порядок фото в карточке
   - Ракурсы съёмки
   - Наличие модели
   - Качество инфографики
   - Наличие видео

   **Цена:**
   - Текущая цена vs наша
   - Участие в акциях
   - СПП на товаре

   **Раскладка по складам:**
   - На каких складах есть товар
   - Сроки доставки
