# Прогноз спроса

Построй прогноз спроса на товар **{{ nmId }}** на {{ days | default: "30" }} дней.

## Алгоритм

### 1. Получи исторические данные

{% if marketplace == "ozon" %}
Используй `ozon_get_orders` за последние 90 дней.
Получи остатки через `ozon_get_stocks`.
{% elsif marketplace == "ym" %}
Используй `ym_get_orders` за последние 90 дней.
Получи остатки через `ym_get_stocks`.
{% else %}
Используй `wb_get_orders` за последние 90 дней.
Получи остатки через `wb_get_inventory`.
{% endif %}

### 2. Агрегируй продажи по неделям

Отфильтруй заказы по товару {{ nmId }}.
Сгруппируй по неделям, рассчитай:
- Среднее за неделю
- Тренд (рост/падение)
- Волатильность (std / mean)

### 3. Выбери метод прогнозирования

| Условие | Метод |
|---------|-------|
| Волатильность < 0.3, нет тренда | Скользящее среднее (7 дней) |
| Есть тренд (рост/падение > 10%) | Взвешенное скользящее среднее |
| Данные > 90 дней + сезонный товар | Сезонный коэффициент |

### 4. Рассчитай прогноз

**Скользящее среднее:**
```
Прогноз_день = Сумма_продаж_7_дней / 7
```

**Взвешенное MA:**
```
Прогноз = (4×Нед1 + 3×Нед2 + 2×Нед3 + 1×Нед4) / 10
```

**Сезонный:**
```
Прогноз = Базовый × Сезонный_коэф_месяца
```

### 5. Учти факторы

Проверь и примени корректировки:
- Участие в акциях WB/Ozon → ×1.5-3.0
- Изменение цены за последний месяц
- Текущий месяц (сезонность категории)
- Out of stock в истории (искажает данные)

### 6. Рассчитай рекомендации по запасам

```
Средние_продажи_день = Прогноз_месяц / 30
Дней_до_обнуления = Текущий_остаток / Средние_продажи_день
Точка_перезаказа = Средние_продажи_день × Срок_поставки × 1.3
Рекомендуемый_заказ = Прогноз_месяц × 1.2 - Текущий_остаток
```

Срок поставки:
- FBO WB: 7-10 дней
- FBS: 3-5 дней
- FBO Ozon: 5-7 дней

## Критерии достоверности

| Уровень | Условия |
|---------|---------|
| **Высокая** | 60+ дней данных, волатильность < 30%, нет out of stock |
| **Средняя** | 30-60 дней данных ИЛИ волатильность 30-50% |
| **Низкая** | < 30 дней данных ИЛИ волатильность > 50% ИЛИ частые out of stock |

## Примечания

- Если товар новый (< 14 дней продаж) — используй аналоги из категории
- Если был out of stock — исключи эти дни из расчёта или укажи в рисках
- Округляй прогнозы до целых чисел
- Диапазон = Прогноз ± 20%
