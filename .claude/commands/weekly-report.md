# Еженедельный отчёт

Комплексный отчёт с использованием всех skills: `wb-unit-economics`, `wb-sales-funnel`, `wb-seo-expert`, `wb-advertising`.

## Интерактивный сбор данных

Используй **AskUserQuestion** для уточнения параметров отчёта:

```
Используй AskUserQuestion с вопросами:

1. "Какие разделы включить в отчёт?"
   - header: "Разделы"
   - multiSelect: true  (можно выбрать несколько)
   - options:
     - "Финансы и выручка (Рекомендуется)" (описание: "Выручка, маржа, прибыль, структура доходов")
     - "Товары и ассортимент (Рекомендуется)" (описание: "Топ продаж, проблемные товары, остатки")
     - "Реклама и ROI" (описание: "Эффективность кампаний, расходы, ROAS")
     - "Отзывы и рейтинг" (описание: "Новые отзывы, средняя оценка, требуют ответа")
     - "Позиции и SEO" (описание: "Изменения позиций по ключевым запросам")

2. "Сравнить с прошлым периодом?"
   - header: "Сравнение"
   - options:
     - "Да, показать динамику (Рекомендуется)" (описание: "Сравним с прошлой неделей, покажем тренды")
     - "Нет, только текущие данные" (описание: "Компактный отчёт без истории")

3. "Добавить план действий на следующую неделю?"
   - header: "План"
   - options:
     - "Да, с приоритетами (Рекомендуется)" (описание: "Конкретные задачи на основе анализа")
     - "Нет, только аналитика" (описание: "Отчёт без рекомендаций")
```

## Алгоритм

1. **Сбор данных за неделю:**
   На основе выбранных разделов получи через API:
   - Продажи (количество, выручка) — `wb_get_orders`
   - Остатки — `wb_get_inventory`
   - Рекламная статистика — `wb_get_campaign_stats`
   - Отзывы — `wb_get_reviews`
   - Воронка продаж — `wb_get_sales_funnel`
