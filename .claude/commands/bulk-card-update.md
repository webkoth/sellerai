# Массовое обновление карточек

Обнови карточки товаров по заданным условиям.

## Параметры

- **Действие:** {{ action }}
- **Значение:** {{ value }}
- **Фильтр:** {{ filter | default: "all" }}
- **Маркетплейс:** {{ marketplace | default: "wb" }}

## Доступные действия

| Действие | Описание | Пример value |
|----------|----------|--------------|
| `add-keyword` | Добавить ключевое слово в название | "женское" |
| `remove-keyword` | Удалить слово из названия | "акция" |
| `replace-text` | Заменить текст | "старый→новый" |
| `update-description` | Обновить описание по шаблону | "template:premium" |
| `add-emoji` | Добавить эмодзи в начало | "🔥" |
| `capitalize` | Привести к Title Case | — |

## Алгоритм

### 1. Получи карточки товаров

{% if marketplace == "ozon" %}
```
ozon_get_product_info — получить карточки
```
{% elsif marketplace == "ym" %}
```
ym_get_products — получить товары
```
{% else %}
```
wb_get_cards — получить карточки
```
{% endif %}

### 2. Примени фильтр

| Фильтр | Логика |
|--------|--------|
| `all` | Все товары |
| `category:Платья` | Только категория |
| `brand:Бренд` | Только бренд |
| `abc:A` | Товары группы A (топ по продажам) |
| `no-keyword:слово` | Товары без ключевого слова |
| `has-keyword:слово` | Товары с ключевым словом |

### 3. Рассчитай изменения

{% if action == "add-keyword" %}
```
Если "{{ value }}" нет в названии:
  Новое_название = Название + " " + "{{ value }}"
```
{% elsif action == "remove-keyword" %}
```
Новое_название = Название.replace("{{ value }}", "").trim()
```
{% elsif action == "replace-text" %}
```
[старый, новый] = "{{ value }}".split("→")
Новое_название = Название.replace(старый, новый)
```
{% elsif action == "update-description" %}
```
Загрузить шаблон {{ value }}
Подставить переменные (название, бренд, характеристики)
```
{% elsif action == "add-emoji" %}
```
Если название не начинается с эмодзи:
  Новое_название = "{{ value }} " + Название
```
{% else %}
```
Новое_название = toTitleCase(Название)
```
{% endif %}

### 4. Проверь ограничения

- Длина названия ≤ 60 символов (WB), ≤ 200 символов (Ozon)
- Нет запрещённых слов (акция, скидка, распродажа)
- Нет дублирования ключевых слов

### 5. Покажи preview

**НЕ применяй изменения автоматически!**

## Примеры использования

### Добавить ключевое слово "летнее" к платьям
```
/bulk-card-update action=add-keyword value=летнее filter=category:Платья marketplace=wb
```

### Удалить слово "акция" из всех названий
```
/bulk-card-update action=remove-keyword value=акция filter=all marketplace=wb
```

### Заменить "женская" на "для женщин"
```
/bulk-card-update action=replace-text value="женская→для женщин" filter=all marketplace=wb
```

### Обновить описания топ-товаров по премиум-шаблону
```
/bulk-card-update action=update-description value=template:premium filter=abc:A marketplace=wb
```

## Безопасность

1. **Всегда показывай preview** перед применением
2. **Проверяй длину названий** — не более лимита маркетплейса
3. **Проверяй запрещённые слова** — акция, скидка, распродажа
4. **Логируй все изменения** в operations.log
