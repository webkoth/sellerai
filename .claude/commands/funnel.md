# Анализ воронки продаж

Используй skill `wb-sales-funnel` для диагностики и рекомендаций.

## Алгоритм

1. **Сбор данных:**
   - Если есть доступ к API/статистике — получи метрики
   - Если нет — запроси у пользователя:
     - Показы за период
     - Клики (переходы на карточку)
     - Добавления в корзину
     - Заказы
     - Выкупы
     - Период анализа

2. **Рассчитай конверсии по этапам:**
   - CTR = Клики / Показы × 100%
   - CR корзина = Корзина / Клики × 100%
   - CR заказ = Заказы / Корзина × 100%
   - % выкупа = Выкупы / Заказы × 100%

3. **Сравни с бенчмарками из skill**

4. **Выведи результат в формате:**

```markdown
## Анализ воронки: [Название товара]

### Период: [дата начала] — [дата конца]

### Текущие показатели

| Этап | Значение | Конверсия | Бенчмарк | Статус |
|------|----------|-----------|----------|--------|
| Показы | X | — | — | — |
| Клики | X | X% (CTR) | 3-7% | 🟢/🟡/🔴 |
| Корзина | X | X% | 5-15% | 🟢/🟡/🔴 |
| Заказы | X | X% | 30-50% | 🟢/🟡/🔴 |
| Выкупы | X | X% | 50-90%* | 🟢/🟡/🔴 |

*зависит от категории
```

## Диагностика по этапам

Используй диагностику из skill `wb-sales-funnel`:

- **Низкий CTR** → Проблема с главным фото, ценой, позицией
- **Мало корзин** → Слабый контент, отзывы, описание
- **Мало заказов** → Долгая доставка, низкое доверие
- **Низкий выкуп** → Несоответствие ожиданиям, качество
