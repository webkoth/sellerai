# Анализ рекламных кампаний

Используй skill `wb-advertising` для анализа и рекомендаций.

## Алгоритм

1. **Сбор данных:**
   Запроси у пользователя статистику рекламы:
   - Тип кампании (Авто/Поиск/Каталог/Карточка)
   - Расход за период
   - Показы
   - Клики
   - Заказы из рекламы
   - Выручка из рекламы

2. **Рассчитай ключевые метрики:**
   - CTR = Клики / Показы × 100%
   - CPC = Расход / Клики
   - CR = Заказы / Клики × 100%
   - CPO = Расход / Заказы
   - ROAS = Выручка / Расход × 100%
   - ROI = (Выручка - Расход) / Расход × 100%

3. **Сравни с бенчмарками из skill**

4. **Выведи результат в формате:**

```markdown
## Анализ рекламы: [Период]

### Общие показатели

| Метрика | Значение | Бенчмарк | Статус |
|---------|----------|----------|--------|
| Расход | X₽ | — | — |
| Показы | X | — | — |
| Клики | X | — | — |
| CTR | X% | 3-7% | 🟢/🟡/🔴 |
| CPC | X₽ | зависит от ниши | — |
| Заказы | X | — | — |
| CR (заказ) | X% | 1-5% | 🟢/🟡/🔴 |
| CPO | X₽ | < маржи | 🟢/🟡/🔴 |
| Выручка | X₽ | — | — |
| ROAS | X% | > 300% | 🟢/🟡/🔴 |
| ROI | X% | > 100% | 🟢/🟡/🔴 |

### Эффективность по кампаниям

| Кампания | Тип | Расход | Заказы | CPO | ROI | Рекомендация |
|----------|-----|--------|--------|-----|-----|--------------|
| [Название] | Авто | X₽ | X | X₽ | X% | 🟢 Оставить |
| [Название] | Поиск | X₽ | X | X₽ | X% | 🟡 Оптимизировать |
| [Название] | Карточка | X₽ | X | X₽ | X% | 🔴 Отключить |

### Диагностика

**Если CTR < 2%:**
- Проблема с карточкой (фото, цена)
- Нерелевантные показы
- Рекомендация: оптимизировать карточку, исключить запросы

**Если ROI < 100%:**
- Реклама убыточна
- Рекомендация: снизить ставки или отключить

**Если ROAS < 300%:**
- Низкая эффективность
- Рекомендация: пересмотреть стратегию
```

## Рекомендации по типам кампаний

Используй стратегии из skill `wb-advertising`:

- **Автореклама** — для старта и тестирования
- **Поиск** — для захвата позиций по ключам
- **Каталог** — для категорий без чёткого поиска
- **Карточка** — для перехвата трафика конкурентов
