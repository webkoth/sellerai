# Анализ воронки продаж Ozon

Проанализируй воронку продаж товара на Ozon.

## Параметры

- **Product ID:** {{ productId | default: "не указан" }}
- **Offer ID:** {{ offerId | default: "не указан" }}
- **Период:** {{ period | default: "30d" }}

## Алгоритм

### 1. Получи данные о товаре

```
ozon_get_product_info productIds=[{{ productId }}]
или
ozon_get_product_info offerIds=[{{ offerId }}]
```

Получи:
- Название товара
- Категория
- Статус модерации
- Видимость (FBO/FBS)
- Ошибки контента

### 2. Получи данные о заказах

```
ozon_get_orders dateFrom=[начало периода] dateTo=[сегодня]
```

Отфильтруй по productId/offerId:
- Всего заказов
- Выполненных (delivered)
- Отменённых (cancelled)
- В процессе (awaiting_*, delivering)

### 3. Получи цены и остатки

```
ozon_get_prices productIds=[{{ productId }}]
ozon_get_stocks productIds=[{{ productId }}]
```

### 4. Рассчитай метрики воронки

**Примечание:** Ozon API не предоставляет данные о показах и кликах напрямую.
Используем косвенные метрики.

```
Заказы_период = количество заказов за период
Выкупы = заказы со статусом delivered
Отмены = заказы со статусом cancelled
Процент_выкупа = Выкупы / Заказы × 100%
Процент_отмен = Отмены / Заказы × 100%
```

**Оценка позиции в категории:**
```
Средний_чек = Сумма_заказов / Количество_заказов
Заказов_в_день = Заказы_период / Дней_в_периоде
```

### 5. Анализ проблем

Проверь:
- ❌ Нет остатков FBO/FBS → товар не продаётся
- ❌ Ошибки модерации → карточка не видна
- ❌ Высокий % отмен → проблема с качеством/доставкой
- ❌ Низкий % выкупа → несоответствие ожиданиям

### 6. Рассчитай упущенную выручку

```
Если отменённых заказов > 10%:
  Упущенная_выручка = Отмены × Средний_чек
```

## Бенчмарки Ozon

| Метрика | Отлично | Хорошо | Требует внимания |
|---------|---------|--------|------------------|
| % выкупа | > 85% | 70-85% | < 70% |
| % отмен | < 5% | 5-15% | > 15% |
| Заказов/день | > 5 | 1-5 | < 1 |
| Content rating | > 80% | 50-80% | < 50% |
| Остатки (дней) | 30-60 | 14-30 | < 14 |

## Ограничения Ozon API

⚠️ **Важно:** Ozon Seller API не предоставляет:
- Количество показов товара
- Количество переходов на карточку
- CTR (Click-Through Rate)
- CR по этапам воронки

Эти данные доступны только в личном кабинете Ozon Seller.

Для полного анализа воронки запроси у пользователя:
1. Показы за период (из ЛК)
2. Клики/переходы (из ЛК)
3. Добавления в корзину (из ЛК)

## Примеры использования

### Анализ по Product ID
```
/ozon-funnel productId=123456789
```

### Анализ по артикулу продавца
```

```

### Анализ за 2 недели
```
/ozon-funnel productId=123456789 period=14d
```
