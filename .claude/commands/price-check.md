# Анализ ценообразования

Используй skills `wb-pricing-strategy` и `wb-unit-economics` для анализа.

## Алгоритм

1. **Получение данных:**
   - Если указан артикул — получи текущую цену через `wb-inventory`
   - Если указана себестоимость — используй для расчётов
   - Если данных нет — используй AskUserQuestion для запроса

2. **Анализ текущей цены:**

   **Структура цены:**
   ```
   Цена продажи = Себестоимость + Комиссия WB + Логистика + Хранение + Реклама + Прибыль
   ```

   **Проверь:**
   - Текущая маржинальность
   - Позиция относительно конкурентов
   - Участие в акциях
   - Текущий СПП

3. **Анализ СПП (Скидка Постоянного Покупателя):**

   | Цена товара | СПП (2024) |
   |-------------|------------|
   | до 500₽ | 0-17% |
   | 500-1000₽ | 17-25% |
   | 1000-2000₽ | 25-30% |
   | 2000₽+ | до 35% |

   **Оптимизация СПП:**
   - Пример: Цена 290₽ → СПП 17% → Покупатель видит 240₽
   - Пример: Цена 320₽ → СПП 25% → Покупатель видит 240₽
   - При одинаковой конечной цене — выгоднее ставить выше!

4. **Рекомендации по стратегии:**

   | Этап | Стратегия | Цена |
   |------|-----------|------|
   | Запуск | Низкая для набора продаж | -10-20% от рынка |
   | Рост | Постепенное повышение | К рыночной |
   | Зрелость | Максимальная прибыль | Рыночная или выше |
   | Спад | Распродажа остатков | Снижение |

## Когда участвовать в акциях

| Ситуация | Участвовать? | Цель |
|----------|--------------|------|
| Запуск товара | ✅ Да | Набор продаж и позиций |
| Много остатков | ✅ Да | Увеличить оборачиваемость |
| Падение позиций | ✅ Да | Вернуться в ТОП |
| Перед поставкой | ✅ Да | Освободить место |
| Конец сезона | ✅ Да | Распродать остатки |
| Пик сезона | ❌ Нет | Максимальная прибыль |
| Мало остатков | ❌ Нет | Сохранить маржу |

## Формула минимальной цены

```
Минимальная цена = Себестоимость / (1 - Комиссия% - Логистика% - Прибыль%)
```

## Предупреждения

- 🔴 Если маржа < 10% — предупредить о риске убытков
- 🟡 Если цена значительно выше конкурентов — предупредить о риске потери продаж
- 🟡 Если СПП можно оптимизировать — показать варианты
