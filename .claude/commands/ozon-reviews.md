# Работа с отзывами Ozon

## Алгоритм

### 1. Определение действия

По умолчанию `action=unanswered` — показать отзывы без ответа.

| action | Что делает |
|--------|------------|
| `list` | Показать все отзывы за период |
| `unanswered` | Показать отзывы без ответа (приоритет негативным) |
| `reply` | Сгенерировать ответ для конкретного отзыва |

### 2. Получение отзывов

Используй `ozon_get_reviews`:

```
ozon_get_reviews({
  dateFrom: "[дата -N дней]",
  dateTo: "[сегодня]",
  status: "unprocessed", // для unanswered
  limit: 50
})
```

### 3. Анализ и приоритизация

Отсортируй отзывы по приоритету:

| Приоритет | Критерий | Действие |
|-----------|----------|----------|
| P1 Критичный | Оценка 1-2, без ответа | Ответить срочно |
| P2 Высокий | Оценка 3, без ответа | Ответить в течение дня |
| P3 Средний | Оценка 4-5, без ответа | Ответить при возможности |
| P4 Низкий | С ответом | Мониторинг |

### 4. Генерация ответа (для action=reply)

**Классификация отзыва:**

| Тип негатива | Признаки | Стратегия ответа |
|--------------|----------|------------------|
| Не по теме (ПВЗ, доставка) | Жалоба на Ozon | Вежливо перенаправить в поддержку Ozon |
| Обоснованная претензия | Реальная проблема с товаром | Извиниться, предложить решение |
| Брак/дефект | Описание дефекта | Предложить связаться для замены |
| Несоответствие ожиданиям | "Думал будет другое" | Уточнить описание, выразить сожаление |

**Генерация ответа:**

Для позитивного отзыва (4-5):
- Поблагодарить
- Выразить радость
- Пригласить снова

Для негативного отзыва (1-2):
- Поблагодарить за обратную связь
- Выразить сожаление (НЕ оправдываться)
- Предложить решение или контакт
- Показать заботу о клиенте

### 5. Формат вывода

**Для action=list или action=unanswered:**

```markdown
## Отзывы Ozon

### Сводка
| Показатель | Значение |
|------------|----------|
| Всего отзывов | X |
| Средняя оценка | X.X |
| Без ответа | X |
| Негативных (1-2) | X |

### Требуют ответа (P1-P2)

| # | Оценка | Товар | Суть | ID |
|---|--------|-------|------|----|
| 1 | ⭐ X | [название] | [краткая суть] | [id] |

### Детали отзывов

---
**⭐ X** | ID: [id] | [дата]
**Товар:** [название]
**Плюсы:** [если есть]
**Минусы:** [если есть]
**Комментарий:** [текст]

💡 **Рекомендация:** [что делать]
---
```

**Для action=reply:**

```markdown
## Ответ на отзыв #[id]

### Анализ

| Параметр | Значение |
|----------|----------|
| Оценка | ⭐ X |
| Тональность | Позитивный / Негативный / Нейтральный |
| Категория проблемы | [если негативный] |

### Исходный отзыв
**Товар:** [название]
**Плюсы:** [если есть]
**Минусы:** [если есть]
> [текст отзыва]

### Рекомендуемый ответ

```
[текст ответа]
```

### Для отправки

Используй команду:
```
ozon_reply_review({ reviewId: [id], text: "[текст]", confirm: true })
```
```

## Правила генерации ответов

1. **Длина:** 2-4 предложения (короткие, ёмкие)
2. **Тон:** Вежливый, профессиональный, без заискивания
3. **Без:** Капса, множественных восклицательных знаков, эмодзи
4. **Персонализация:** По возможности обратиться к конкретике отзыва

## Шаблоны-основы

**Позитивный (4-5):**
```
Благодарим за отзыв! Рады, что [конкретика из отзыва].
Будем рады видеть вас снова!
```

**Негативный (1-2) — качество:**
```
Благодарим за обратную связь. Приносим извинения за доставленные неудобства.
Мы обязательно учтём ваши замечания. Для решения вопроса свяжитесь с нами через сообщения в личном кабинете.
```

**Негативный — не по теме:**
```
Спасибо за отзыв. К сожалению, [описанная проблема] не относится к нашему товару —
это зона ответственности Ozon. Рекомендуем обратиться в поддержку площадки.
```

**Нейтральный (3):**
```
Благодарим за отзыв и рекомендации! Мы учтём ваши пожелания для улучшения товара.
Будем рады видеть вас среди наших покупателей снова!
```

## После анализа

- Если негативный отзыв указывает на реальную проблему — предложи обновить карточку через `ozon_update_product`
- Если типичная претензия повторяется — предложи добавить информацию в описание
- Подведи итог: сколько отзывов требуют срочного ответа
