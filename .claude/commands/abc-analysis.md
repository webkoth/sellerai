# ABC-анализ ассортимента

Используй skills `wb-unit-economics` и `wb-sales-funnel` для комплексного анализа.

## Интерактивный сбор данных

Используй **AskUserQuestion** для уточнения параметров анализа:

```
Используй AskUserQuestion с вопросами:

1. "По какой метрике провести ABC-анализ?"
   - header: "Метрика"
   - options:
     - "Выручка (Рекомендуется)" (описание: "Стандартный ABC-анализ по объёму продаж")
     - "Прибыль" (описание: "Учитывает маржинальность, требует данные о себестоимости")
     - "Количество продаж" (описание: "По числу проданных единиц")

2. "За какой период анализировать?"
   - header: "Период"
   - options:
     - "7 дней" (описание: "Краткосрочная картина, актуальные тренды")
     - "30 дней (Рекомендуется)" (описание: "Оптимальный баланс актуальности и репрезентативности")
     - "90 дней" (описание: "Долгосрочный анализ, сглаживает сезонность")

3. "Добавить XYZ-анализ стабильности спроса?"
   - header: "XYZ"
   - options:
     - "Да, ABC+XYZ матрица (Рекомендуется)" (описание: "Комбинированный анализ для оптимизации запасов")
     - "Нет, только ABC" (описание: "Базовый анализ без оценки стабильности")
```

## Алгоритм

1. **Сбор данных:**
   - Получи список товаров через skill `wb-inventory`
   - Используй параметры из AskUserQuestion для настройки анализа:
     - Метрика: выручка / прибыль / количество
     - Период: 7 / 30 / 90 дней
     - XYZ-анализ: да / нет

2. **Расчёт ABC-классификации:**
   - Отсортируй товары по выбранной метрике (убывание)
   - Рассчитай накопительный % от общей суммы
   - Присвой категории:
     - **A** — до 80% накопительного итога
     - **B** — от 80% до 95%
     - **C** — от 95% до 100%

3. **Дополнительный XYZ-анализ (если есть история):**
   - **X** — стабильный спрос (коэфф. вариации < 10%)
   - **Y** — умеренные колебания (10-25%)
   - **Z** — непредсказуемый спрос (> 25%)

- Расширять ассортимент в этой нише

## Принцип Парето

ABC-анализ основан на принципе 80/20:
- 20% товаров приносят 80% выручки (категория A)
- Фокус на категории A даёт максимальный ROI усилий
