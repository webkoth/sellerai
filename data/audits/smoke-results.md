## ⚠️ S0: Предусловие — регистрация новых subagent_type

**Обнаружено при попытке Smoke 1:**
```
Error: Agent type 'operations-director' not found.
Available agents: ads-optimizer, claude, ..., marketplace-analyst, ..., review-manager, ...
```

**Причина:** harness Claude Code регистрирует subagent_type из `.claude/agents/*.md` **только при старте сессии**. Новые агенты `operations-director` и `external-traffic-manager`, созданные в текущей сессии, не доступны как subagent_type до перезапуска.

**Что это означает:**
- Полные функциональные smoke-сценарии Smoke 1–5 в обычном виде требуют **перезапуска Claude Code в каталоге проекта**.
- В текущей сессии возможна только **логическая валидация** через `general-purpose` агента с подменой роли (передаём содержимое нашего агента как часть prompt'а).

**Действие:** в этом отчёте — логическая валидация. После перезапуска оператор должен прогнать сценарии заново через настоящие `/audit` и `/external-deal` и обновить статусы ниже.
