## ⚠️ Конфликты направлений

### Конфликт 1: «Запустить активную оптимизацию рекламы» vs «Нет доступа к API + остатки=0»
- **Позиции:** ads-optimizer запросил перевыпуск токена и активацию. marketplace-analyst указывает на stock=0.
- **Разрешение:** реклама БЕЗ остатков = слив бюджета. **Сначала** восстановить остатки (🔴 #1), **параллельно** перевыпустить токен (🟡 #2). Активная оптимизация ставок — только в Неделю 3-4, после поставки и пополнения карточек на FBO.

### Конфликт 2: «Дозаполнить характеристики быстро» vs «Нет данных рекламы для приоритизации SKU»
- **Позиции:** content-optimizer хочет покрыть все 15 карточек. ads-optimizer мог бы подсказать, какие SKU дают максимальный показ → приоритет.
- **Разрешение:** ICE-Score характеристик высокий (120 000), Ease=4 → делаем все 15 за один заход без ожидания данных рекламы. Это дешевле, чем разбивать на этапы.

### Конфликт 3: «Маржа неподтверждена» vs «Действовать сейчас»
- **Позиции:** marketplace-analyst не может валидировать ROI рекламы без СБ. operations-director рекомендует действовать по 🔴 уже сейчас.
- **Разрешение:** действия 🔴 #1-3 не требуют знания СБ (поставка, контент, отмены). СБ нужна только для блока рекламы в Неделе 3-4 — к этому времени данные должны быть получены.
